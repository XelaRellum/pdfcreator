PDFCreator Hilfe
 
 

Profile verwenden

Sie können mehrere Druckprofile erstellen. Jedes Profil speichert Einstellungen zur Ausgabedatei und was damit geschieht (Drucken nach dem Speichern, Aktionen ausführen etc.).

Um mehr über die Einstellungen zu erfahren, schauen Sie bitte in die anderen Abschnitte dieses Kapitels.

Profile verwalten

Sie können Profile neu anlegen, umbennen oder löschen, indem Sie den (+)-Button, Umbenennen-Button oder (x)-Button benutzen. Profile können auch in eine Datei gespeichert oder daraus geladen werden. So können Sie Einstellungen sichern, übertragen oder mit anderen Benutzern austauschen.

Anmerkung: Es ist nicht möglich ein Profil zu löschen, das noch mit einem Drucker verbunden ist. Sie müssen die Zuordnung aufheben, bevor das Profil gelöscht werden kann.
