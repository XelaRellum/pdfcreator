PDFCreator Hilfe
 
 

Nach der Installation von PDFCreator bekomme ich den Fehler "Exception EInOutError in module is-XXXXX.exe at 0000631C. File not found".

Der Fehler entsteht durch eine nicht vollständig durchgeführte InstallShield Installation. In den meisten Fällen kann die angegebene Datei bedenkenlos gelöscht werden. Wenn Sie sich nicht sicher sind, ob die Datei gelöscht werden kann, kann diese auch umbenannt oder verschoben werden.
