PDFCreator Hilfe
 
 

Programm-Schriftarten

Die Einstellungen legen die Schriftart fest, die PDFCreator für alle Texte und Felder im Programm benutzt. Normalerweise wird die Standardschriftart von Windows benutzt, aber es kann jede beliebige Schrift eingestellt werden. Mit dem Knopf "Testen" kann die Schrift vorher einmal angeschaut werden, ohne sie ganz zu übernehmen.
