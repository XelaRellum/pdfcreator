PDFCreator Hilfe
 
 

Das Gradzeichen ragt in nachfolgende Zeichen hinein.

Das Problem wird dadurch verursacht, dass in der Anwendung eine Standardschriftart verwendet wird (z.B. Arial), die im Drucker durch eine Systemschrift ersetzt wird. Diese ist aber nicht komplett identisch, insbesondere das Gradzeichen ist anders und ragt dadurch eventuell in die nachfolgenden Zeichen rein. Das kann man verhindern, indem auch Standardfonts in den Druckauftrag eingebettet werden:

In den Druckereinstellungen unter Erweitert befindet sich eine Baumansicht. Dort unter Grafik -> TrueType-Schriftarten steht "Durch Geräteschriftart ersetzen" Wenn das auf "Als Softfont in den Drucker laden" ersetzt wird, funktioniert es wie gewünscht.
