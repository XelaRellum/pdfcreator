TIFF-Einstellungen

Dieser Menüpunkt erlaubt Einstellungen für das TIFF-Format vorzunehmen. Das TIFF-Format kann für Mehrseiten-Dokumente verwendet werden.

Auflösung: Einstellen der DPI eines Bildes. Je höher die DPI Zahl ist, um so detailierter, aber auch um so größer wird das Bild. Der Standardwert ist 150.

Farben: Hier kann die Anzahl der Farben ausgewählt werden. Die beste und höchstmögliche Auswahl sind 24 Bit Farben.
