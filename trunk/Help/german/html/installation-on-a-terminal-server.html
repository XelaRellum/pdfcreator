PDFCreator Hilfe
 
 

Installation auf einem Terminal Server

Um PDFCreator auf einem Terminal Server zu installieren, müssen folgende Schritte durchgeführt werden.

1.) In der Kommandozeile in den Installationsmodus wechseln : "change user /install"
2.) Das Setup mit dem Parameter /expert starten und "Standardinstallation" auswählen.
3.) Nach Beendigung des Setups zurück in den Anwendungsmodus wechseln: "change user /execute"
4.) Das war alles :-)
