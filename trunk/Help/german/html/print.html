PDFCreator Hilfe
 
 

Drucken

Hier kann festgelegt werden, ob nach dem Speichern das Dokument auf einem Drucker ausgedruckt werden soll.

Drucken nach dem Speichern: Aktiviert das Drucken nach dem Speichern.

Drucker: Auswahl des Druckers.
Beispiele:
"HP4300"
"\\printserver\HP4300"

Bits je Pixel: Setzt die Farbauflösung für das Drucken. Mögliche Werte sind True Color (volle Farbauflösung), CMYK und Mono (Schwarz\Weiß).

Benutzereingriff: Legt fest, ob dem Benutzer ein Druckerdialog angezeigt werden soll.
Möglich Werte sind:

  • Aus (Standard)
  • Zeige den Standard-Druckerdialog
  • Zeige den Druckerinstallationsdialog
  • Verwende den Standarddrucker ohne eine Benutzerauswahl

Verstecke den Verarbeitungsdialog während des Druckens: Ist diese Einstellung aktiviert, kann die Anzeige des Druckfortschrittes aktiviert bzw. deaktiert werden.

Zweiseitiger Druck: Aktiviert den zweiseitigen (Duplex) Druck. Zusätzlich kann ausgewählt werden ob, über die Lange Kante (Standard) oder über die Kurze Kante gedruckt werden soll.

Setze maximale Druckauflösung: Ermöglicht es die Auflösung für das Drucken festzulegen. Der Standardwert ist 600. Eine hohe Druckauflösung bedeuted einen besseren Ausdruck aber auch größere Druckdateien.
