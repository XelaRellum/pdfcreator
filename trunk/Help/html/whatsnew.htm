What's New in PDFCreator

Version 0.8.1 (November 4, 2004)

    Improvements:
  • Using GPL-Ghostscipt 8.15
    Bug Fixes:
  • Missing msmapi32.ocx is included.
  • Improve email support.
  • Terminalserver should work
  • Using a modifies ghostpdf.ppd
  • fixed english.ini
  • Fixed some minor bugs.
    Kwown issues:
  • no hyperlinks possible

Version 0.8.0 (April 2, 2004)

    Improvements:
  • Now it is possible to donate us. (See: http://www.PDFCreator.de.vu/)
  • Security! PDFCreator can handle either security settings from AFPL Ghostscript above 8.14 or can use then program 'pdfenc' from Steven Lee.
  • Improved printer driver handling
  • 2 new variables: OptionsEnable and OptionsVisible (a simple protections to save the options - useful for administrators)
  • Set this variables manually in the file PDFCreator.ini to save the options.
  • Email Clients: Succesfully tested with MS Outlook, Outlook Express, Thunderbird, Eudora, Pegasus Mail (WsendTo.exe), The Bat!, Foxmail, Sylpheed-Claws, IncrediMail. Problems with: Ak-Mail, GeMail, Postguard, Vivian Mail
  • Add PDF document properties 'Keywords' and 'Subject'
  • You can add printable files (also via 'Drag and Drop') and through the windows explorer context menu.
  • Added Commandline parameter PF: Convert a printable file in a pdf document. Example: PDFCreator.exe -PF"c:\autoexec.bat"
  • Update check from PDFCreator Menu
    Bug Fixes:
  • Fixed a problem using email function
  • Fixed a problem using combine function
  • Fixed some minor bugs
    Kwown issues:
  • Doesn't work in remote sessions on terminalservers (Please patient, we are working hard on this problem.)
  • No hyperlinks possible

Version 0.7.1 (July 22, 2003)

    Bug Fixes:
  • Fixed up a problem using email function

  • Fixed up a problem using non-conform postscript files

  • Command line parameter: -CLEARCACHE to clear the temp. cache

Version 0.7.0 (July 17, 2003)

    What's new:
  • 2 new output formats: Postscript (PS) and Encapsulated Postscript (EPS)

  • Process priority can be changed. (only for NT and above)

  • Enhanced error handling

  • Improved program speed

  • Individual installation of Ghostscript

  • Ask if .ps files should be associated with PDFCreator

  • Command line parameter: -IF"My Inputfile.ps" -OF"My Outputfile.pdf"

  • Special filename handling include filename substitutions

  • Switch back to Ghostscript 8.00

  • Support the Outlook Format: Plain Text

  • New test page
    Bug Fixes:
  • Fixed up the 'subst' problem in the 'Save as' dialog

  • Fixed up the problem that the program can't create a temp directory

  • Fixed up some localization errors

  • Fixed up some problems in the setup

  • Fixed up the problem on computers with low resources

  • (PDFCreator does not create pdfs with many pages.)

  • Fixed up 'Laufzeitfehler 9' when start PDFCreator

  • Fixed up uninstall printer driver

  • Can handle special drives and partitions
