Après l'installation, j'obtiens un message d'erreur "Exception EInOutError in module is-XXXXX.exe at 0000631C. File not found". Que faire?

C'est un problème lié aux installations antérieures réalisées à l'aide de InstallShield. Dans la plupart des cas, vous pouvez sans risque supprimer de votre répertoire Windows le fichier dont le nom est donné. Si vous n'êtes pas sûr de son importance, déplacez-le dans un répertoire de votre choix pour le sauvegarder.
