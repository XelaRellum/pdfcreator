Aide PDFCreator
 
 

Paramètres d'enregistrement

Le menu Enregistrement permet de définir la manière dont PDFCreator définit le nom des fichiers qu'il crée. Voici la liste des options de ce menu :

Nom de fichier : Permet de changer la façon dont PDFCreator nommera les fichiers qu'il crée. Par defaut, le nom du PDF est celui du document source suiivi de l'auteur s'il existe.

Substitution dans les noms de fichiers : Permet de définir la façon dont les fichiers issus de certains programmes sont nommés au moment de leur conversion. Par exemple, lorsqu'ils seront convertis en PDF les documents Microsoft Word seront nommés filename.doc.pdf. Vous pouvez supprimer la mention .doc du nom de fichier automatiquement en ajoutant une option de subsitution qui remplacera ".doc" par "" (ou par une chaîne vide).

Format d'enregistrement standard : Cette option, définit le format de fichier par défaut dans les boîtes de dialogues pour l'enregistrement.
