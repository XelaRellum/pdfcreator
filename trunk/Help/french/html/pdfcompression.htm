Paramètres PDF : Compression

Ce menu ajuste les paramètres de compression utilisés pour la création du PDF.

Comprimer les objets Texte : Lorsque l'option est utilisée, PDFCreator compacte le texte dans le document pour réduire la taille du fichier.

Compression des images : Comme pour les objets Texte, les images peuvent être compactées pour réduire la taille du PDF. Trois types d'images peuvent être compressées : images couleur, images en niveaux de gris et images monochromes. Voici les options de compression des images :

  • Type de compression : Règle le type de compressqion utilisé pour réduire la taille des images. Plusieurs types de compression peuvent être utilisés, y compris JPEG et ZIP.
  • Rééchantillonner : Définit comment l'image est recalculée lors de la compression. Le rééchantillonnage réduit le nombre de pixels de l'image, donc le nombre de pixels utilisés est réduit également. Le rééchantillonnage réduit la taille de tout fichier image.
  • Résolution : Fixe le nombre de ppi, ou points par pouce, que l'image aura dans le PDF.
