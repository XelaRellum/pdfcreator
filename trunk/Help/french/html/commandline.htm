Aide PDFCreator
 
 

Paramètres de ligne de commande

PDFCreator vous permet de créer des documents via l'interface de ligne de commande. Voici les différentes commandes acceptées par PDFCreator :

/CheckInstance
Si vous utilisez cette option, PDFCreator affiche une boîte de dialogue contenant des informations sur les instances en cours d'exécution (PDFCreator, PDFSpooler et Transtool). Vous pouvez associer cette option avec l'option /NoStart. Si vous voyez la propriété "Local", c'est qu'il existe une instance locale. Regarder sur le bureau, la barre de tâches ou le gestionnaire de tâches pour localiser l'instance locale. Si vous voyez seulement la propriété "Global", alors il existe au moins une instance dans une des sessions de vos serveurs. Cette option est utile pour le débogage et pour les administrateurs afin de savoir si une instance de PDFCreator s'exécute sur un serveur.
Exemple : PDFCreator.exe /CheckInstance

/ClearCache
Supprime tous les fichiers temporaires de PDFCreator. Cela est utile si PDFCreator ne peut démarrer à cause d'un fichier temporaire endommagé.
Exemple : pdfcreator.exe /ClearCache

/DeleteIF
Efface le fichier source après l'avoir converti. Cette option doit être utilisée avec l'option /IF.
Exemple : pdfcreator.exe /IF"C:\description.ps" /OF"C:\description.pdf" /DeleteIF

/IF<input-filename> /OF<output-filename>
Convertit un fichier postscript ou bitmap (JPEG, BM) en un fichier PDF ou bitmap. Le paramètre /OF doit être utilisé avec le paramètre /IF. Pour définir le format de sortie, incluez l'extension de fichier correspondante dans le nom du fichier de sortie. Il n'y a PAS d'espace entre les options et les noms de fichiers.
Le paramètre /OutputSubFormat définit le sous-format. Habituellement "PDF/A-1b" et ""PDF/X" peuvent être utilisés pour le format PDF.
Exemples:
pdfcreator.exe /IF"C:\description.ps" /OF"C:\description.pdf" /OutputSubFormat"PDF/A-1b"
pdfcreator.exe /IF"C:\description.ps" /OF"C:\description.tif"
pdfcreator.exe /IF"C:\image1.jpg" /OF"C:\image1.pdf"
pdfcreator.exe /IF"C:\image1.bmp" /OF"C:\image1.png"

/NoAbortIfRunning
Ouvre une nouvelle instance de PDFCreator sans vérifier si le programme est déjà en cours d'exécution. Il est utile de combiner cette option avec l'option /NoProcessing.
Exemple : pdfcreator.exe /NoAbortIfRunning

/NoProcessing
Lorsque PDFCreator est lancé avec l'option /NoProcessing , l'ensemble des documents envoyés à PDFCreator ne sont pas convertis. Ils sont stockés dans la file d'attente jusqu'à ce que PDFCreator soit lancé sans l'option /NoProcessing.
Exemple : pdfcreator.exe /NoProcessing

/NoProcessingAtStartup: Lorsque vous utilisez cette option, PDFCreator ne convertira pas immédiatement les documents . Vous pouvez lancer manuellement la conversion en allant dans le menu Document et en sélectionnant Imprimer, ou en décochant la ligne Arrêt dans le menu Imprimante. L'option est utile si vous voulez collecter plusieurs fichiers d'abord, puis les fusionner entre eux en vue de créer un seul pdf.
Propriétés :
0 - Off (Defaut)
1 - On

/NoStart
Lorsque vous utilisez l'option /PF, PDFCreator reste ouvert une fois sa tâche terminée. L'option /NoStart ferme automatiquement PDFCreator à la fin du traitement.
Exemple : pdfcreator.exe /NOSTART /PF"C:\help.doc"

/OpenOF
Ouvre le fichier converti à la fin du traitement. Cette option ne s'utilise qu'avec les options /IF et l'option /OF.
Exemple: pdfcreator.exe /IF"C:\description.ps" /OF"C:\description.pdf" /OpenOF

/OptionsFile<option-filename>
Précise les paramètres particuliers que doit utiliser PDFCreator.
Exemple : pdfcreator.exe /OptionsFile"C:\MyPDFCreator.ini"
(Nota: Avec cette option (et un peu de travail) vous pouvez utiliser plus d'une imprimante par poste.)

/PF<filename>
Imprime un fichier à l'aide du programme par défaut, déterminé par l'extension du fichier. En général, cette option est utile lorsqu'on utilise le mode Enregistrement automatique. Cette option ne peut pas être utilisée avec l'option /OF. Il n'y a PAS d'espace entre l'option et le nom de fichier.
Exemple : pdfcreator.exe /PF"C:\help.doc"
Vous pouvez également utiliser les caractères génériques pour créer des PDF à partir de tous les fichiers d'un dossier particulier.
Exemple: pdfcreator.exe /PF"C:\*.doc"

/ShowOnlyLogfile
Ouvre seulement le fichier journal de PDFCreator. Aucune autre fenêtre n'est ouverte.
Exemple: pdfcreator.exe /ShowOnlyLogfile

/ShowOnlyOptions
Ouvre seulement le panneau de configuration de PDFCreator. Aucune autre fenêtre n'est ouverte.
Exemple: pdfcreator.exe /ShowOnlyOptions
