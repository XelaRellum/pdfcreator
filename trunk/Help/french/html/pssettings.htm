Aide PDFCreator
 
 

Paramètres PS

Ce menu ajuste les paramètres pour la conversion au format Postscript (PS). Le format Postscript est un format standard pour l'impression et les images.

Niveau : Règle le niveau de language Postscript, ou la version, à utiliser pour convertir un document. Si vous créez un document PS en vue d'une impression, l'imprimante doit préciser quel niveau de langage Postscript elle peut utiliser. Le niveau par défaut est 2.
