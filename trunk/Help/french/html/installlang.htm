Aide PDFCreator
 
 

Installer un fichier de traduction

PDFCreator possède déjà un grand nombre de fichiers de traduction accessibles sur son site Web. Pour installer un fichier de traduction déjà réalisé, enregistrez simplement le fichier dans le répertoire language du répertoire d'installation de PDFCreator sur votre disque dur. Au prochain lancement de PDFCreator, la langue sera accessible dans le menu Langue du moniteur d'impression.
