Aide PDFCreator
 
 

Qu'est-ce qu'un fichier de traduction?

Un fichier de traduction est un fichier spécial qui contient tous les textes qui vous seront affichés lors de l'utilisation de PDFCreator. Cela permet facilement de changer la langue du programme, ainsi un grand nombre de personnes peut utiliser le programme. Les fichiers de traduction peuvent être facilement créés en utilisant l'outil de traduction (Translation Tool) fourni avec PDFCreator.
