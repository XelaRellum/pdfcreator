Créer un PDF

Créer un PDF avec PDFCreator ne peut être plus facile. Suivez simplement les étapes suivantes et vous créerez un PDF en un rien de temps. Il y a cinq manières de créer un PDF avec PDFCreator : Imprimer, Glisser/Déplacer, Ajout de document, Ligne de commande, et depuis l'explorateur Windows.

Méthode 1: Imprimer

1. Ouvrez le document que vous souhaitez convertir en PDF dans son application d'origine.

2. Imprimez le document comme vous le feriez avec une imprimante classique.

3. Lorsque la fenêtre de dialogue pour l'impression s'affiche, sélectionnez l'imprimante PDFCreator à la place de l'imprimante par défaut. Si l'imprimante PDFCreator est déjà votre imprimante par défaut, ne changez rien. Ensuite, cliquez sur OK.

4. Une autre fenêtre apparaît alors, et vous demande de saisir plusieurs informations pour créer votre PDF. Cliquez sur le bouton "Enregistrer" pour choisir l'endroit où vous souhaitez enregistrer votre PDF ou cliquez sur le bouton "Mettre en file d'attente", pour créer le PDF plus tard.

    Note : C'est aussi à cet endroit que vous pouvez spécifier dans quel format enregistrer le fichier. vous pouvez enregistrer au format PDF, PNG, JPEG, BMP, PCX, TIFF, PS, et EPS.

5. Si vous sélectionnez "Enregistrer", votre PDF sera créé et ouvert ensuite dans votre logiciel de lecture des fichiers PDF par défaut. Si vous avez sélectionné "Mettre en file d'attente", le moniteur d'impression de PDFCreator s'ouvrira et le document apparaîtra dans la file d'attente.

Méthode 2: Glisser/Déplacer

1. Ouvrez le moniteur d'impression de PDFCreator.

2. Cliquez sur le document que vous voulez convertir et faites le glisser sur le moniteur d'impression. Si PDFCreator n'est pas votre imprimante par défaut, il vous demandera de le définir par défaut temporairement pour effectuer la tâche. L'imprimante par défaut sera rétablie une fois la conversion terminée.

3. Une autre fenêtre apparaîtra, vous demandant de saisir des informations pour la création de votre PDF. Cliquez sur le bouton "Enregistrer" pour choisir l'endroit où vous souhaitez enregistrer le PDF ou cliquez sur le bouton "Mettre en file d'attente" pour créer le PDF plus tard.

    Note : C'est aussi à cet endroit que vous pouvez spécifier dans quel format enregistrer le fichier. Vous pouvez enregistrer au format PDF, PNG, JPEG, BMP, PCX, TIFF, PS, et EPS.

4. Si vous sélectionnez "Enregistrer", votre PDF sera créé et ouvert ensuite dans votre logiciel de lecture des fichiers PDF par défaut. Si vous avez sélectionné "Mettre en file d'attente", le moniteur d'impression de PDFCreator s'ouvrira et le document apparaîtra dans la file d'attente.

Méthode 3: Ajout de document

1. Ouvrez le moniteur d'impression PDFCreator .

2. Cliquez sur le menu Document et sélectionnez Ajouter.

3. dans la liste de sélection des types de fichiers, sélectionnez "Tous les fichiers" puis sélectionnez le document que vous voulez convertir en PDF. Si PDFCreator n'est pas votre imprimante par défaut, il vous demandera de le définir par défaut temporairement pour effectuer la tâche. L'imprimante par défaut sera rétablie une fois la conversion terminée.

Méthode 4: Ligne de commande

1. Allez dans le menu Démarrer et cliquez sur Exécuter.

2. Dans la boîte Ouvrir tapez : PDFCreator.exe /PF"Chemin du fichier à convertir" (Saisissez le chemin complet vers PDFCreator.exe et ne mettez pas d'espace entre /PF et le chemin). Vous pouvez aussi placer cette commande dans un raccourci, un fichier batch ou dans le prompt DOS/Windows.

3. Si PDFCreator n'est pas votre imprimante par défaut, il vous demandera de le définir par défaut temporairement pour effectuer la tâche. L'imprimante par défaut sera rétablie une fois la conversion terminée.

4. L'application par défaut associée au document sera ouverte et imprimera automatiquement le document avant de se refermer. PDFCreator s'ouvrira alors et une fenêtre apparaîtra, vous demandant de saisir des informations pour la création de votre PDF. Cliquez sur le bouton "Enregistrer" pour choisir l'endroit où vous souhaitez enregistrer le PDF ou cliquez sur le bouton "Mettre en file d'attente" pour créer le PDF plus tard.

    Note : C'est aussi à cet endroit que vous pouvez spécifier dans quel format enregistrer le fichier. Vous pouvez enregistrer au format PDF, PNG, JPEG, BMP, PCX, TIFF, PS, et EPS.

4. Si vous sélectionnez "Enregistrer", votre PDF sera créé et ouvert ensuite dans votre logiciel de lecture des fichiers PDF par défaut. Si vous avez sélectionné "Mettre en file d'attente", le moniteur d'impression de PDFCreator s'ouvrira et le document apparaîtra dans la file d'attente.
