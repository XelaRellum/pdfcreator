Paramètres EPS

Ce menu ajuste les paramètres pour la conversion au format EPS (Encapsulated Postscript). Le format EPS est un format standard pour l'impression et les images.

Niveau : Règle le niveau de language EPS, ou la version, à utiliser pour convertir un document. Si vous créez un document EPS en vue d'une impression, l'imprimante doit préciser quel niveau de langage EPS elle peut utiliser. Le niveau par défaut est 2.
