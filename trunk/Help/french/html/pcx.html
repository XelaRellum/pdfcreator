Aide PDFCreator
 

PCX

Ce menu permet d'ajuster les paramètres pour la conversion en fichier image au format PCX (Portable Network Graphics). Le format PCX ne doit être utilisé que pour les documents d'une page.

Résolution : Règle la résolution (dpi ou points par pouce) de l'image. Plus la résolution est élevée, plus l'image est détaillée, et plus le fichier correspondant est volumineux. La résolution par défaut est 150 dpi.

Couleurs : Régle le nombre de couleurs qui doivent être utilisées pour la conversion en PCX. Le nombre maximal de couleurs, qui correspond à la plus grande qualité, est de 32 bits CMJN.
