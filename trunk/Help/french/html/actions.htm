Aide PDFCreator
 
 

Actions

Dans Actions vous pouvez spécifier les programmes et les scripts qui sont executés avant ou après l'enregistrement de PDFCreator.

Le format interne est PostScript. Les programmes qui souhaitent modifier les fichiers avant que PDFCreator ne les convertisse doivent être capables de manipuler ce format. Les programmes intervenant après la conversion reçoivent le PDF final.

Attendre que l'application/le script soit prêt(e) : Cette option définit si PDFCreator attend ou non que le programme soit prêt. Cette option est requise si le programme modifie le fichier avant que PDFCreator ne le convertisse.

Paramètres de l'application : Ces paramètres sont passés à l'application/au script. Classiquement les variables <TempFilename> pour le nom du fichier temporaire créé avant l'enregistrement et <OutputFilename> pour le PDF une fois enregistré. Les noms de fichiers doivent être entourés de guillemets doubles pour éviter les problèmes liés aux espaces dans les noms de fichier.

Style de fenêtre : Cette option permet de définir l'apparence de la fenêtre dans laquelle le programme s'exécute avant ou après l'enregistrement :
Cachée : Le programme s'exécute de façon invisible.
Normal/Activée : Le programme s'exécute normalement en avant-plan.
Réduite/Activée : Le programme s'exécute dans une fenêtre réduite en avant-plan.
Agrandie/Activée : Le programme est exécuté dans une fenêtre agrandie et en avant-plan.
Normal/non active : Le programme s'exécute normalement en arrière-plan.
Réduite/non active : Le programme s'exécute dans une fenêtre réduite en arrière-plan.
