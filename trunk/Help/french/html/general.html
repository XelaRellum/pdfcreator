Aide PDFCreator
 
 

Général

L'onglet Général permet à l'utilisateur d'ajuster les paramètres de base pour la création d'un PDF.

Compatibilité : Détermine quelle version du format PDF est utilisé pour convertir les documents en PDF. Cela correspond à la version d'Adobe® Acrobat® capable de lire le PDF.

Rotation automatique : Lorsque cette option est utilisée, les pages horizontales sont pivotées dans le sens vertical, et les pages verticales sont pivotées dans le sens horizontal. Vous pouvez choisir de pivoter la totalité du document ou une seule page.

Surimpression : La surimpression est utilisée pour imprimer une couleur au-dessus des autres sans mélange. Lors de la création d'un PDF, vous pouvez activer ou non la surimpression. Lorsque la surimpression est désactivée, cela correspond à la valeur Non-nulle, lorsque elle est activée, cela correspond à la valeur Totale.

Mode d'affichage des pages : Précise la manière dont les différentes pages d'un document sont affichées : une par une, par deux, et si le défilement peut être utilisé.

Mode d'affichage par défaut : Précise comment ouvrir le fichier PDF. Les vignettes doivent-elles être affichées ou le document doit-il être ouvert en plein écran etc.

Optimisé pour le web : Lorsque cette option est utilisée, PDFCreator optimise le PDF créé pour un affichage plus rapide sur le web. Au lieu que le PDF soit téléchargé totalement pour pouvoir s'afficher, il s'affichera progressivement en fonction du téléchargement. Le cryptage ne peut pas être utilisé en même temps que cette option. Si vous utilisez le cryptage, le PDF ne sera pas optimisé.
