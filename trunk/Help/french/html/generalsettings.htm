Aide PDFCreator
 
 

Paramètres généraux

La page paramètres généraux concerne les réglages de base de l'application PDFCreator.

Imprimer la page de test : Crée une page PDF d'exemple pour tester l'imprimante PDFCreator et ses pilotes. Cela permet à l'utilisateur de créer un PDF sans avoir à utiliser un autre programme pour le test. C'est particulièrment utile en cas de dysfonctionnement de PDFCreator sur votre système.

Priorité de traitement : La priorité de traitement détermine la quantité de puissance de l'ordinateur utilisée pour créer les PDF. Selon le réglage, la rapidité avec laquelle les PDF seront créés augmentera ou diminuera. C'est utile sur un système avec des ressources faibles.

Ne pas imprimer la file d'attente au démarragede PDFCreator : Si vous activez cette option, PDFCreator ne démarrera aucune impression. Vous pouvez lancer manuellement la conversion en allant dans le menu Document et en sélectionnant Imprimer, ou en décochant la ligne Arrêt dans le menu Imprimante. Cette option est utile si vous souhaitez rassembler plusieurs documents avant de les fusionner et de créer un PDF.

Ne pas confirmer la définition de PDFCreator comme imprimante par défaut : Occasionnellement, PDFCreator doit être défini comme imprimante par défaut pour accomplir certaines tâches. Lorsqu'elle est activée, PDFCreator se définit lui-même comme imprimante par défaut pour accomplir son travail. Sinon, PDFCreator vous demande une confirmation avant de le faire lorsque c'est nécessaire.

Méthode d'envoi des e-mails : Cette option permet à l'utilsiateur de choisir la façon dont sont envoyés les e-mails.
Valeurs :
Automatique - PDFCreator détecte le client de messagerie par défaut et détermine la meilleure méthode (Défaut)
Interface MAPI - PDFCreator utilise l'interface MAPI.
Utiliser sendmail.dll - PDFCreator utilises l'extension shell "Send To/Mail Recipient".

Associer PDFCreator avec les fichiers Postscript : Lorsque vous cliquez sur ce bouton, PDFCreator devient l'application par défaut pour ouvrir les fichiers Postscript (.ps) sous Windows. Ne pas cliquer si vous souhaitez utiliser un autre programme par défaut pour ces fichiers.

Couleur des boîtes de dialogue : Avec ce réglage, vous pouvez changer la couleur des cadres de boîtes de dialogue. Cela peut être utile pour les utilisateurs qui ont les yeux sensibles ou une vue déficiente.
Valeurs :
1 - Rouge et bleu dégradés (Défaut)
2 - Rouge et bleu simple

Afficher une animation durant le traitement : Ce réglage supprime l'animation affichée durant la création d'un PDF ou d'un bitmap. Par défaut l'animation est activée.

Intégrer PDFCreator aux menus contextuels : Lorsque ce bouton est activé PDFCreator est intégré à Windows. Lorsque vous faites un clic droit sur un document, une nouvelle ligne apparaîtra dans le menu contextuel intitulée "Créer des fichiers PDF ou bitmap avec PDFCreator". Cele permet à l'utilisateur de convertir rapidement des documents en PDF sans avoir à ouvrir le document. Notez que l'usage de cette fonction requiert que PDFCreator soit temporairement défini comme imprimante par défaut .

Supprimer l'intégration aux menus contextuels : Lorsque on clique, PDFCreator n'apparaît plus dans les menus contextuels de Windows.
