Installation on a Terminal Server

If you want install PDFCreator on a terminalserver do this:

1.) In a command box switch to the installation mode with "change user /install"
2.) Start the setup with the switch /Expert and choose "Standardinstallation"
3.) If the setup has finnished switch back to the application mode with "change user /execute"
4.) That's all. :-)
