PDFCreator Help
 
 

Settings in PDFCreator

Since version 0.8.1 RC8, PDFCreator saves all settings in the registry. If you choose the "Standard installation" PDFCreator saves settings to the following registry location: HKCU\Software\PDFCreator

If your operating system is Windows 2000 and above, the setup saves some settings for the default user in the following registry location: HKU\.DEFAULT\Software\PDFCreator

In a "Server installation" of PDFcreator, the settings are saved to the following registry location: HKLM\Software\PDFCreator

While the default location for settings is in the registry, it is also possible to store the settings in the the old .ini files mode. In this case PDFCreator save all settings to ini files. It is also possible to add an external ini file as default file. For more information use the commandswitch /? on the setup program.
Attention: There exists some problems with server based profiles and if the application data path is on a network drive. Because of this case server installations can't use the old ini-mode.

If you choose the "Standard installation" and are saving to ini files, the settings are saved to the application data path of user:
Example: C:\Documents and Settings\[user profile]\Application Data\PDFCreator

If your operating system is Windows 2000 and above, the setup saves some settings for the default user in the following directory:
Example: C:\Documents and Settings\Default User\Application Data\PDFCreator

In a "Server installation" of PDFCreator, settings are saved in the followind directory:
C:\Documents and Settings\All User\Application Data\PDFCreator

Procedure of reading the settings

On Windows 2000 and above, PDFCreator reads the settings from the default user first. If there are no settings, PDFCreator will use its internal default settings. Next, PDFCreator reads the settings from the user. Only the user settings PDFCreator finds will overwrite the settings from the default user. Finally, PDFCreator looks for machine settings in
HKLM\Software\PDFCreator
or
C:\Documents and Settings\All User\Application Data\PDFCreator.
Only the machine settings PDFCreator finds will overwrite the settings from the user. With this behaivour it is possible for administrators to define settings for all users.

Procedure of saveing the settings

If you choose the "Standard installation" of PDFCreator, settings are saved to either of the following:
HKCU\Software\PDFCreator
or
C:\Documents and Settings\[user profile]\Application Data\PDFCreator

If you choose the "Server installation" of PDFCreator, settings are saved to either of the following:
HKLM\Software\PDFCreator
or
C:\Documents and Settings\All User\Application Data\PDFCreator

Hint:  If you need to use settings other than the default, use the /REGFile= or /INIfile= parameters on the installation program. For more information on the parameters that can be used on installation program, use the /? parameter.
