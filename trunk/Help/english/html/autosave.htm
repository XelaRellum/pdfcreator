Auto-Save Mode

Auto-Save Mode allows the user to automatically create PDF's with a predefined settings and save it to a predefined location. When Auto-Save Mode is enabled PDFCreator does not ask for a filename and a location. Below is the list of settings that must be set for Auto-Save Mode:

  • Auto-Save format: The save format is the file type that you want PDFCreator to save the new document as.
  • Filename: Sets the file name that you want all documents you create to have. the <DateTime> variable inputs the creation date and time as the file name.
  • Filename-Token: The filename-token is another variable that is added to the documents file name.
  • Directory: An optional feature is to set a different directory to automatically save the documents.

With Auto-Save Mode you can choose whether you want to automatically save to a certain directory or let PDFCreator save to the desktop. Note: You can use PDF security features in Auto-Save Mode by using the PDFOwnerPasswordString and PDFUserPasswordString hidden settings. For more information on the hidden settings, click here.
