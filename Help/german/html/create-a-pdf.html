PDFCreator Hilfe
 

Eine PDF-Datei erstellen

Eine PDF-Datei mit PDFCreator zu erstellen könnte kaum leichter sein. Es gibt fünf verschiedene Wege PDF-Dateien zu erstellen: Drucken, Drag and Drop, Dokument hinzufügen, Kommandozeile und über den Windows Explorer.

Drag and Drop

Windows Explorer

Methode 1: Drucken

1. Öffnen Sie das zu konvertierende Dokument in Ihrer gewohnten Anwendung.

2. Drucken Sie die Datei, als würden Sie sie auf einem normalen Drucker ausdrucken.

3. Wenn der Drucken-Dialog erscheint, wählen Sie den Drucker PDFCreator aus. Wenn PDFCreator bereits der Standarddrucker ist, muss nichts geändert werden.

4. Ein weiteres Fenster erscheint, wo Sie mehrere Informationen zur PDF-Datei angeben können. Drücken Sie auf die Schaltfläche "Speichern" um die PDF-Datei jetzt zu erstellen oder auf "Warten", um das zu einem späteren Zeitpunkt zu tun.

    Hinweis: Hier können Sie auch das Ausgabeformat für das Dokument bestimmen. Sie haben die Wahl zwischen PDF, PNG, JPEG, BMP, PCX, TIFF, PS und EPS.

5. Wenn sie "Speichern" gewählt haben, wird die PDF-Datei erstellt und in der Standardanwendung für PDF-Dateien geöffnet.

Methode 2: Drag and Drop

1. Öffnen Sie den PDFCreator Druckermonitor.

2. Ziehen Sie die zu konvertierende Datei mit Hilfe der Maus dort hinein. Wenn PDFCreator nicht ihr Standarddrucker ist, wird er fragen, ob er sich vorübergehend als Standarddrucker einrichten soll, damit der Vorgang abgeschlossen werden kann. Der alte Standarddrucker wird wieder aktiviert, sobald das Dokument erstellt wurde.

3. Ein weiteres Fenster erscheint, wo Sie mehrere Informationen zur PDF-Datei angeben können. Drücken Sie auf die Schaltfläche "Speichern" um die PDF-Datei jetzt zu erstellen oder auf "Warten", um das zu einem späteren Zeitpunkt zu tun.

    Hinweis: Hier können Sie auch das Ausgabeformat für das Dokument bestimmen. Sie haben die Wahl zwischen PDF, PNG, JPEG, BMP, PCX, TIFF, PS und EPS.

.4. Wenn sie Speichern gewählt haben wird die PDF erstellt und in der Standardanwendung für PDF-Dateien geöffnet.

Methode 3: Dokument hinzufügen

1. Öffnen Sie den PDFCreator Druckermonitor.

2. Wählen Sie aus dem Menü Dokument den Punkt Hinzufügen.

3. Wählen sie als Dateityp "Alle Dateien" und wählen dann die zu konvertierende Datei aus. Wenn PDFCreator nicht ihr Standarddrucker ist, wird er fragen, ob er sich vorübergehend als Standarddrucker einrichten soll, damit der Vorgang abgeschlossen werden kann. Der alte Standarddrucker wird wieder aktiviert, sobald das Dokument erstellt wurde.

4. Ein weiteres Fenster erscheint, wo Sie mehrere Informationen zur PDF-Datei angeben können. Drücken Sie auf die Schaltfläche "Speichern" um die PDF-Datei jetzt zu erstellen oder auf "Warten", um das zu einem späteren Zeitpunkt zu tun.

    Hinweis: Hier können Sie auch das Ausgabeformat für das Dokument bestimmen. Sie haben die Wahl zwischen PDF, PNG, JPEG, BMP, PCX, TIFF, PS und EPS.

5. Wenn sie "Speichern" gewählt haben, wird die PDF-Datei erstellt und in der Standardanwendung für PDF-Dateien geöffnet.

Methode 4: Kommandozeile

1. Wählen Sie aus dem Startmenü den Punkt Ausführen.

2. Im Textfeld Öffnen geben Sie ein: PDFCreator.exe /PF"Pfad zur Datei" (Geben Sie den vollständigen Pfad zu PDFCreator.exe ein und beachten Sie, dass keine Leerzeichen zwischen /PF und dem Pfad stehen). Sie können diesen Befehl auch in eine Verknüpfung, Batchdatei oder den DOS-Prompt eingeben.

3. Wenn PDFCreator nicht ihr Standarddrucker ist, wird er fragen, ob er sich vorübergehend als Standarddrucker einrichten soll, damit der Vorgang abgeschlossen werden kann. Der alte Standarddrucker wird wieder aktiviert, sobald das Dokument erstellt wurde.

4. Ein weiteres Fenster erscheint, wo Sie mehrere Informationen zur PDF-Datei angeben können. Drücken Sie auf die Schaltfläche "Speichern" um die PDF-Datei jetzt zu erstellen oder auf "Warten", um das zu einem späteren Zeitpunkt zu tun.

    Hinweis: Hier können Sie auch das Ausgabeformat für das Dokument bestimmen. Sie haben die Wahl zwischen PDF, PNG, JPEG, BMP, PCX, TIFF, PS und EPS.

5. Wenn sie "Speichern" gewählt haben, wird die PDF-Datei erstellt und in der Standardanwendung für PDF-Dateien geöffnet.

Methode 5: Windows Explorer

1. Klicken sie mit der rechten Maustaste auf eine Datei und wählen Erzeuge PDF mit PDFCreator.

2. Wenn PDFCreator nicht ihr Standarddrucker ist, wird er fragen, ob er sich kurz als Standarddrucker einrichten kann, damit der Vorgang abgeschlossen werden kann. Der alte Standarddrucker wird wieder eingerichtet, sobald das Dokument erstellt wurde.

3. Ein weiteres Fenster erscheint, wo Sie mehrere Informationen zur PDF-Datei angeben können. Drücken Sie auf die Schaltfläche "Speichern" um die PDF-Datei jetzt zu erstellen oder auf "Warten", um das zu einem späteren Zeitpunkt zu tun.

    Hinweis: Hier können Sie auch das Ausgabeformat für das Dokument bestimmen. Sie haben die Wahl zwischen PDF, PNG, JPEG, BMP, PCX, TIFF, PS und EPS.

4.

Wenn sie "Speichern" gewählt haben, wird die PDF-Datei erstellt und in der Standardanwendung für PDF-Dateien geöffnet.
