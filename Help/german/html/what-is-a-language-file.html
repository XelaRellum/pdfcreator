Was ist eine Sprachdatei?

Eine Sprachdatei ist eine spezielle Textdatei, in der alle Texte enthalten sind, die von PDFCreator benötigt werden. Dies macht die Verwendung von unterschiedlichen Sprachen in PDFCreator sehr einfach. Sprachdateien können mit dem beiliegenden Programm TransTool erstellt und bearbeitet werden.
