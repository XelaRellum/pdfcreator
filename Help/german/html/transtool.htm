Das Übersetzungsprogramm TransTool

Das Übersetzungsprogramm TransTool ermöglicht eine bessere Internationalisierung von PDFCreator. Somit ist ein einfacher Wechsel in eine gewünschte Sprache möglich. Einige Leistungsmerkmale des Übersetzungsprogrammes TransTool sind:

  • Übersetzen von einer beliebigen Sprache in eine beliebige Sprache und speichern der Sprachdatei für eine spätere Verwendung.
  • Verändern vorhandener Sprachdateien, um diese nach eigenen Bedürfnissen anzupassen.
  • Eine umfangreiche Suchfunktion.
  • Schnelles finden von noch leeren Einträgen.

Um mehr über das Übersetzungsprogramm TransTool zu erfahren, klicken Sie auf Was ist eine Sprachdatei? and Erstellen eine Sptrachdatei.
