Setup Fehler: PDFCreator ist bereits installiert

Es kann vorkommen, dass eine Installation nicht möglich ist, da dass Setup meldet das PDFCreator bereits installiert ist. In diesem Fall wurde PDFCreator nicht vorschriftsmäßig deinstalliert bzw. bei der Deinstallation trat ein Fehler auf. Es wird dann die Fehlermeldung:

Das Programm ist bereits installiert.
Für eine Neuinstallation deinstallieren Sie zuvor das Programm.
Die Installation wird abgebrochen.

angezeigt.

Es kann auch nicht ausgeschlossen werden, dass geänderte Kopien des PDFCreators im Umlauf sind, die denselben Setup-Quelltext verwenden. Dies ist durchaus möglich, da die Quelltexte des PDFCreator offen gelegt sind.

Um dieses Problem zu beheben gibt es den Setup Parameter /ForceInstall, der eine Installation über eine bereits bestehende Installation erzwingt.

Um das Setup mit diesem Parameter aufzurufen, kann entweder die Kommandozeile verwendet oder eine Verknüpfung erstellt werden.
