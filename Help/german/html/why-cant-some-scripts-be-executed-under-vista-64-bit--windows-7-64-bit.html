Warum können einige Skripte unter Vista (64 bit)\ Windows 7 (64 bit) nicht ausgeführt werden?

VB-Skripte werden unter Vista (64 bit)\ Windows 7 (64 bit) als 64-bit-Skripte ausgeführt.
Einige DLLs sind aber 32-bit-DLLs. Das Skript muss entweder mit der 32-bit von WScript.exe aus dem Ordner %SystemRoot%\SysWOW64 ausgeführt werden oder es wird global der Wert des Registry-Schlüssels:


HKEY_CLASSES_ROOT\VBSFile\Shell\Open\Command
(Standard) von

"%SystemRoot%\System32\WScript.exe" "%1" %*

auf

"%SystemRoot%\SysWOW64\WScript.exe" "%1" %*

umgestellt.
