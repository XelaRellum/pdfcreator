PDF-Einstellungen: Unterschreiben

Mit Hilfe eines Zertikates ist es möglich eine PDF-Datei elektronisch zu unterschreiben. Dazu sind verschiedene Voraussetzungen nötig:

  1. Es ist ein elektronisches Zertifikat in Form einer PFX\P12-Datei nötig, dass zum Unterschreiben geeignet ist.
  2. .Net 1.1 muss installiert sein. Dies bedeutet, dass unter Windows 95 und Windows NT 4 Workstation kein Unterschreiben möglich ist, da .Net 1.1 für diese Betriebssysteme nicht verfügbar ist.
  3. Die pdfforge.dll muss im PDFCreator-Programmpfad unter PlugIns\pdfforge vorhanden sein. Ist .Net 1.1 installiert, wird diese DLL automatisch bei der Installation von PDFCreator mit installiert.

Nachfolgend eine Beschreibung der Parameter:

  • PFX\P12-Datei: Hier muss eine PFX\P12-Datei angegeben werden.
  • Unterschriftsgrund, Unterschriftskontakt, Unterschriftsort: Hier können Grund, Kontakt und Ort für die Unterschrift vorgegeben werden.
  • Unterschriftsposition: Es ist möglich die Unterschrift im Dokument durch ein zusätzliches Bild anzeigen zu lassen. Mit diesen Einstellungen kann angegeben werden, wo das Bild angezeigt werden soll.
  • Mehrfache Unterschriften erlauben: Sollen mehrere Unterschriften möglich sein, muss diese Option aktiviert sein.
