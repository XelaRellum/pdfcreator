Ghostscript Einstellungen

Die Ghostscript Einstellungen legen fest, welche Version von Ghostscript benutzt wird.

Ghostscript Version: Listet die installierten Ghostscript-Versionen auf. Standard ist die interne Version von PDFCreator. Sie können aber auch andere von Ihnen installierte Versionen hier auswählen.

Zusätzliche Ghostscriptparameter: Mit Hilfe dieser Einstellung können zusätzlich Ghostscriptparameter festgelegt werden. Zum Trennen der Parameter muss das Pipe Symbol | verwendet werden. Standardmäßig ist Wert leer.
Beispiel:
AdditionalGhostscriptParameters=-dTextAlphaBits=4|-dGraphicsAlphaBits=4|-dDOINTERPOLATE

Zusätzlicher Ghostscript-Suchpfad: Diese Einstellung erlaubt zusätzliche Verzeichnisse zum Ghostscript-Suchpfad hinzuzufügen. Dies kann verwendet werden, um zusätzliche Verzeichnisse mit Schriftarten hinzuzufügen.

Windows Schriftarten verwenden: Wird diese Einstellung aktiviert, durchsucht Ghostscript das Windows-Schriftarten Verzeichnis nach möglichen Schriftarten.
Eigenschaften:
0 - Aus
1 - Ein (Standard)
