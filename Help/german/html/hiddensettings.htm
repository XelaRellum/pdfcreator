PDFCreator Hilfe
 
 

Versteckte Einstellungen

PDFCreator hat einige Einstellungen, die für Testzwecke gedacht sind. Standardmäßig werden die Einstellungen in der Registry gespeichert. Es ist auch möglich die Einstellungen in einer Datei PDFCreator.ini zu speichern. Soll eine INI-Datei verwendet werden, muss das Programm neu installiert werden und dabei der Parameter /UseINI verwendet werden. Näheres dazu unter Einstellungen.

ClientComputerResolveIPAddress: In speziellen Fällen ist es möglich, dass der Token <ClientComputer>eine IP-Adresse an Stelle eines Hostnamen enthält. Wird diese Einstellung auf 1 gesetzt, versucht PDFCreator die IP-Adresse in einen Hostnamen aufzulösen.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

DisableEmail: Diese Einstellung deaktiviert den Email-Button in PDFCreator.
Eigenschaften:
0 - Email-Button ist aktiviert (Standard)
1 - Email-Button ist deaktviert

DeviceWidthPoints, DeviceHeightPoints: Diese beiden Einstellungen legen eine fest vorgegebene Breite und Höhe für das zu verwendene Papierformat fest. Die Einheit ist 1/72 eines Zolls und kann als Alternative für die Einstellung Papersize verwendet werden, aber nicht in Kombination damit.
Eigenschaften:
Der Standardwert ist -1 und bedeuted, dass PDFCreator diese Einstellung ignoriert.

NoProcessingAtStartup: Wird diese Einstellung aktiviert, beginnt PDFCreator nach dem Start nicht mit der Verarbeitung. Die Verarbeitung kann manuell gestartet werden, wenn über das Menü Dokument der Menüpunkt Drucken gewählt wird oder durch die Deaktivierung des Menüpunktes Drucker anhalten im Menü Drucker. Diese Einstellung ist sinnvoll, wenn zunächst Dokumente im Druckmonitor gesammelt und diese dann später verbunden bzw. einzeln verarbeitet werden sollen.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

OnePagePerFile: Die Einstellung erlaubt Benutzern eine neue Datei für jede Seite des Originals zu erstellen. Aufgrund von Einschränkungen in Ghostscript funktioniert das nicht mit PDF und EPS Dateien.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

Papersize: Diese Einstellung kann verwendet werden, um eine Standardpapiergröße vorzugeben.
Eigenschaften:
Standardmäßig ist dieser Wert leer. Ansonsten können die in der Ghostscript Dokumentation angegebenen Papergrößen verwendet werden.

Nachfolgend sind einige mögliche Papierformate aufgelistet:
11x17, ledger, legal, letter, lettersmall, archE, archD, archC, archB, archAa0, a1, a2, a3, a4, a4small, a5, a6, a7, a8, a9, a10, isob0, isob1, isob2, isob3, isob4, isob5, isob6, c0, c1, c2, c3, c4, c5, c6

(http://www.ghostscript.com/doc/AFPL/8.50/Use.htm#Known_paper_sizes)

PDFCompressionColorCompression*, PDFCompressionGreyCompression*: Mit Hilfe dieser Parameter lassen sich individuelle Kompressionsfaktoren für die einzelnen Kompressionsgrade vorgeben, um so die Komprimierung von Farb- und Graustufenbildern besser steuern zu können. Die Werte müssen größer 0 sein.
Standardwerte:
PDFCompressionColorCompressionJPEGMaximumFactor=2
PDFCompressionColorCompressionJPEGHighFactor=0.9
PDFCompressionColorCompressionJPEGMediumFactor=0.5
PDFCompressionColorCompressionJPEGLowFactor=0.25
PDFCompressionColorCompressionJPEGMinimumFactor=0.1
PDFCompressionGreyCompressionJPEGMaximumFactor=2
PDFCompressionGreyCompressionJPEGHighFactor=0.9
PDFCompressionGreyCompressionJPEGMediumFactor=0.5
PDFCompressionGreyCompressionJPEGLowFactor=0.25
PDFCompressionGreyCompressionJPEGMinimumFactor=0.1

PDFOptimize: Die Einstellung optimiert die erstellte PDF-Datei für die Darstellung im Web. Dadurch kann die erste Seite bereits angezeigt werden, bevor das gesamte Dokument geladen ist.
Zur Zeit ist es nicht möglich eine Verschlüsselung und Optimierung gleichzeitig zu nutzen. Verschlüsselte PDF-Dateien sind nicht für die schnelle Web-Anzeige optimiert.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

PDFOwnerPasswordString: Diese Einstellung kann in Verbindung mit den Sicherheitseinstellungen von PDF Dateien benutzt werden. PDFCreator übernimmt das Passwort automatisch als Besitzerpasswort. ACHTUNG: Der Text in der PDFCreator.ini ist nicht verschlüsselt. Sie Benutzen diese Option auf eigene Gefahr.

PDFUserPasswordString: Diese Einstellung kann in Verbindung mit den Sicherheitseinstellungen von PDF Dateien benutzt werden. PDFCreator übernimmt das Passwort automatisch als Benutzerpasswort. ACHTUNG: Der Text in der PDFCreator.ini ist nicht verschlüsselt. Sie Benutzen diese Option auf eigene Gefahr.

PrintAfterSaving: Druckt automatisch ein Dokument nach dem Speichern.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

PrintAfterSavingPrinter: Ist die Einstellung PrintAfterSaving aktiviert, kann hier der Name des Druckers vorgegeben werden. Wird diese Einstellung leer gelassen oder existiert der Drucker nicht, erscheint ein Druckerauswahl-Dialog. Es kann ein lokaler Drucker wie auch ein Netzwerkdrucker ausgewählt werden. Dabei ist immer der Präfix "\\spool\" zu verwenden. Standardmäßig ist diese Einstellung leer.
Beispiele:
PrintAfterSavingPrinter="\\spool\HP4300"
PrintAfterSavingPrinter="\\spool\\\printserver\HP4300"

PrintAfterSavingQueryUser: Ist die Einstellung PrintAfterSaving aktiviert, kann mit dieser Einstellung vorgegeben werden, ob dem Benutzer ein Druckerdialog angezeigt werden soll.
Eigenschaften:
0 - Aus (Standard)
1 - Zeigt den Standard-Druckerdialog
2 - Zeigt den Installations-Druckerdialog
3 - Wählt den Standarddrucker ohne eine Benutzerauswahl.

PrintAfterSavingNoCancel: Ist die Einstellung PrintAfterSaving aktiviert, kann mit Hilfe dieser Einstellung die Anzeige des Druckfortschrittes aktiviert bzw. deaktiert werden.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

PrintAfterSavingDuplex: Ist die Einstellung PrintAfterSaving aktiviert, kann mit dieser Einstellung der Duplexdruck, falls vom Drucker unterstützt, aktiviert werden.
Eigenschaften:
0 - Kein Duplexdruck (Standard)
1 - Verwendet Duplexdruck, falls vom Drucker unterstützt

PrintAfterSavingTumble: Ist die Einstellung PrintAfterSaving und die Einstellung PrintAfterSavingDuplex aktiviert, kann mit dieser Einstellung festgelegt werden, an welcher Kante der Duplexdruck zu erfolgen hat.
Eigenschaften:
0 - Lange Kante (Standard)
1 - Kurze kante

RemoveAllKnownFileExtensions: Wird diese Einstellung aktiviert, entfernt PDFCretor alle bekannten Erweiterungen von rechts beginnend, bis eine unbekannte Erweiterung gefunden wurde. Beispiel: Wenn der interne Titel einer Postscriptdatei 'Pic.jpg.-12.31.2005.pcx' lautet, veränert PDFCreator diesen zu 'Pic.jpg.-12.31.2005'. Dies erleichert das Speichern wesentlich.
Eigenschaften:
0 - Aus
1 - Ein (Standard)

RunProgramAfterSaving: Diese Einstellung legt fest, ob das unter RunProgramAfterSavingProgramname angegebene Programm ausgeführt wird, wenn PDFCreator das Dokument gespeichert hat. Der Name Dokumentes wird als Parameter übergeben.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

RunProgramAfterSavingProgramname: Hier wird der Pfad zu dem auszuführenden Programm festgelegt.
Beispiel: RunProgramAfterSavingProgramname="C:\Programme\Mein Programm\Mein Programm.exe"

RunProgramAfterSavingProgramParameters: Diese Einstellung wird in Verbindung mit RunProgramAfterSaving benutzt. Sie gibt die Parameter an, mit denen das Programm aufgerufen wird.

RunProgramAfterSavingWaitUntilReady: Diese Einstellung legt fest, ob PDFCreator wartet, bis das Programm die Verarbeitung des Dokumentes abgeschlossen hat.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

RunProgramAfterSavingWindowstyle: Diese Einstellung legt die Erscheinung des Fensters des Programmes fest, das nach dem Speichern ausgeführt wird.
Eigenschaften:
0 - Das Programm wird versteckt ausgeführt.
1 - Das Programm wird normal ausgeführt und aktiviert.
2 - Das Programm wird minimiert ausgeführt und aktiviert.
3 - Das Programm wird maximiert ausgeführt und aktiviert.
4 - Das Programm wird normal ausgeführt und nicht aktiviert.
6 - Das Programm wird minimiert ausgeführt und nicht aktiviert.

RunProgramBeforeSaving: Diese Einstellung legt fest, ob das unter RunProgramBeforeSavingProgramname angegebene Programm ausgeführt wird, bevor PDFCreator das Dokument gespeichert hat. PDFCreator wartet, bis das Programm die Verarbeitung des Dokumentes abgeschlossen hat. Der Name Dokumentes wird als Parameter übergeben.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

RunProgramBeforeSavingProgramname: Hier wird der Pfad zu dem auszuführenden Programm festgelegt.
Beispiel: RunProgramBeforeSavingProgramname="C:\Programme\Mein Programm\Mein Programm.exe"

RunProgramBeforeSavingProgramParameters: Diese Einstellung wird in Verbindung mit RunProgramBeforeSaving benutzt. Sie gibt die Parameter an, mit denen das Programm aufgerufen wird.

RunProgramBeforeSavingWindowstyle: Diese Einstellung legt die Erscheinung des Fensters des Programmes fest, das vor dem Speichern ausgeführt wird.
Eigenschaften:
0 - Das Programm wird versteckt ausgeführt.
1 - Das Programm wird normal ausgeführt und aktiviert.
2 - Das Programm wird minimiert ausgeführt und aktiviert.
3 - Das Programm wird maximiert ausgeführt und aktiviert.
4 - Das Programm wird normal ausgeführt und nicht aktiviert.
6 - Das Programm wird minimiert ausgeführt und nicht aktiviert.

SendEMailAfterAutoSaving: Ermöglich das Versenden einer Email nach dem automatischen Speichern einer Datei.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

SendMailMethod: Legt das Verfahren fest wie eine Email erstellt werden soll.
Eigenschaften:
0 - Das Programm wählt automatisch die beste Methode für das Versenden einer Email. (Standard)
1 - Das Programm verwendet die Mapi-Schnittstelle.
2 - Das Programm verwendet die "SendTo"-Methode des Windows-Explorers.

ShowAnimation: Diese Einstellung ermöglicht die Animation während des Erstellungsprozesses aus- oder anzuschalten.
Eigenschaften:
0 - Zeigt keine Animation.
1 - Zeigt eine Animation. (Standard)

StampFontColor: Legt die Farbe für den Stempeltext fest. Verwenden Sie dafür HTML-Farben.
Beispiel für Rot: StampFontColor=#FF0000

StampFontname: Legt die Schriftart für den Stempeltext fest.

StampFontsize: Legt die Schriftgröße für den Stempeltext fest.

StampOutlineFontthickness: Legt die Breite der Umrandung für den Stempeltext fest.

StampString: Wenn diese Einstellung gesetzt ist, legt PDFCreator einen Stempel über das Dokument. Sie können die Schriftart, Farbe, Größe und Umrandung des Stempels mit den folgenden Einstellungen festlegen. Diese Einstellung ist noch sehr experimentell. Bitte kontrollieren Sie die erstellten Dokumente.

StampUseOutlineFont: Diese Einstellung legt fest, ob eine Umrandung für den Stempeltext benutzt wird.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

StandardCreationdate: Diese Einstellung erlaubt dem Benutzer, ein Standarderstellungsdatum für alle Dokumente anzugeben.

StandardDateformat: Diese Einstellung erlaubt dem Benutzer, das Format für das Erstellungs- und Änderungsdatum festzulegen.

StandardKeywords: Diese Einstellung erlaubt dem Benutzer, Schlüsselwörter für alle Dokumente anzugeben.

StandardModifydate: Diese Einstellung erlaubt dem Benutzer, ein Standarddatum der letzten Änderung für alle Dokumente anzugeben.

StandardSaveformat: Diese Einstellung erlaubt das Standardformat im Speichern unter Dialog festzulegen. Das Standardformat ist pdf. Die folgenden Formate sind erlaubt: pdf, png, jpg, bmp, pcx, tif, ps, eps.
Beispiel:
StandardSaveformat=tif

StandardSubject: Diese Einstellung erlaubt dem Benutzer, einen Standardbetreff für alle Dokumente anzugeben.

StandardTitle: Diese Einstellung erlaubt dem Benutzer, ein Standardtitel für alle Dokumente anzugeben.
