The Translation Tool

The Translation Tool enables you to better internationalize PDFCreator by giving the user the ability to change the language in PDFCreator. Some features of the Translation Tool are:

  • Translate PDFCreator from any language to any language and save it to language files for later use.
  • Modify existing language files to suit your needs.
  • Search the different keys and template text equivalents.
  • Quickly find the remaining blank keys in your language file.

To learn more about the Translation Tool, click on the What is a Language File? and Create a Language File articles.
