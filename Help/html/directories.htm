Directories Properties

The directories properties window allows the user to change the directories where PDFCreator accesses or stores information and files. By default, PDFCreator sets these directories for you, but you may change them to fit your system better. It is not recommended to change these directories. Here is a list of what each directory connects to:

Temporary Files: The directory where PDFCreator temporally creates files when making documents.
