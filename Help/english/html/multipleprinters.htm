PDFCreator Help
 
 

Using Multiple Printers

PDFCreator allows you to have multiple printers. In the Main Window, you can access the printer settings via Printer -> Printers.

Add/Delete a printer

You can use PDFCreator to add new printers. These printers are no ordinary printers, but designed for the usage with PDFCreator. Just type in a printer name and click on "Add printer". After selecting a printer, you can remove it by clicking on "Del printer". This will add/remove real Windows printers.

Note: You need to have Administrator privileges to add or delete printers. On Windows Vista and above, you also have to run PDFCreator as Administrator.

Link a printer to a profile

You can link each printer with an option profile. You could have one printer for invoices that saves the invoice as PDF to a certain folder and another one for standard usage.

Other possibilities are:

  • a JPEG image printer
  • high res PDFs for printing
  • low res PDFs for eMail
  • a PDF printer with predefined Author, Security Settings etc

To learn how to use profiles, please have a look at the "Using Profiles" Section.
