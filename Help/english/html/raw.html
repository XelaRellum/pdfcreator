RAW

This menu changes the settings when converting to the RAW image format. This format can be useful when you want to directly access the raster data.

Resolution: Sets the dpi or dots per inch of the image. The higher the dpi the more detailed the image will be and the larger the file size will be. The default dpi is 150.

Colors: Sets the amount of colors that should be used when converting to RAW format. The best and highest setting is 24 bit color.
