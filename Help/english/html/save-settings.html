Save Settings

The Save settings menu allows the user to change the way PDFCreator names the files it creates. Below is a list of options in the Save settings menu:

Filename: The filename option is where you change the way you want PDFCreator to name the documents it creates. The default is to name PDF's by the source document's title and then author, if it exists.

Filename Substitution: If you don't want to change the filename settings for every document you create but you want to change they way files created in a certain program are saved, you can use filename substitution. For example, some Microsoft Word documents when converted into PDF will be named filename.doc.pdf. You can remove the .doc from the filename automatically by adding a filename substitution to replace ".doc" with "" (or nothing).

Standard save format: This setting sets the standard save format for the save dialogs.

Allow Ghostscript escape character in filenames: If this setting is activated, the Ghostscript character % is allowed in filenames. If this setting is deactivated, the character is used to control the single page output.
