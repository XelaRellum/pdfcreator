PDFCreator Help
 

Print

If you use this setting PDFCreator prints a a document after saving.

Print after saving: Activate print after saving.

Printer: Select which printer to use.
Examples:
"HP4300"
"\\printserver\HP4300"

Bits per Pixel: Set the resolution for printing. Possible values are True Color (full colors), CMYK and Mono (monochrome).

Query user: This setting enables the user to show or hide a printer dialog.
Possible values are:

  • Off (Default)
  • Show the standard printer dialog
  • Show the printer setup dialog
  • Select the default Windows printer without any user interaction

Hide the progress dialog during printing: This setting shows or hides the progress dialog.

Duplex: Activate duplex. Additional you can choose to use tumble or not.

Set maximum resolution: Allows to set the resolution for printing. The default alue is 600. A higher resolution can produce better output but also increases the size of the printed files.
