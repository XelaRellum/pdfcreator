Fonts

The Fonts menu changes the settings for embedding and subsetting fonts into a PDF.

Embed all Fonts: When checked, all fonts that were used in the source document are embedded into the PDF. This allows the PDF to be opened anywhere and still be viewed correctly. This is a recommended option.

Subset Fonts: When embedding fonts, if the entire font is not used you can choose to subset a portion of it. If you chose to subset fonts, only the characters used will be embedded into the PDF. This will reduce the file size of the PDF. When checked, you set the percentage of characters the font must below before it starts to subset the font.
