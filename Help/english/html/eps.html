EPS

This menu changes the settings when converting to EPS format, or Encapsulated Postscript format. The EPS format is a standard for printing and imaging.

Language Level: Sets the language level or version to use when converting a document. If you are making a EPS file to be used for printing the printer may specify which level of EPS it can handle. The default Language Level is 2.
