PCL

This menu changes the settings when converting to the PCL format. PCL is a standard for HP printers.

Resolution: Set the DPI of the image. The higher this value is, the better quality is, but this also increases the file size. The default value is 150.

Colors: Sets the amount of colors that should be used when converting to PCL format. The best and highest setting is 24 bit color.
