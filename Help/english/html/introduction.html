PDFCreator Help
 
 

Introduction

You can access the help menu by clicking the Help command on the Help (?) menu in the print monitor. You can also reach it by hitting F1 on your keyboard anywhere in the program.

Here is a brief overview on how to navigate this help menu:

  • Select the Contents tab to browse the help menu by category.
  • Select the Index tab to find topics that are similar to your keyword search.
  • Select the Search tab to search for a particular topic.
  • Select the Favorites tab to save your favorite topics for quick and easy access.
