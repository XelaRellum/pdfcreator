Aide PDFCreator
 
 

Répertoires

Les options de répertoires permettent de définir les répertoires où PDFCreator accède où il stocke informations et fichiers. Il n'est pas recommandé de changer ces répertoires.

Fichiers temporaires : C'est le répertoire où PDFCreator enregistre temporairement les fichiers qu'il crée. Il est possible d'utiliser les variables PDFCreator et les variables d'environnement, mais soyez prudents avec ces options. Toutes les variables n'ont pas de sens pour cette option. Il est recommandé de n'utiliser que les suivantes : <Temp>, <Redmon_User>, <Redmon_Machine>, <Clientcomputername> et <Computername>. Si vous utilisez les variables d'environnement, veuillez n'utiliser que les variables système et redémarrez votre système après avoir défini une variable d'environnement système.

Exemple:
1.) Réglez la variable d'environnement système PTPath: PTPath=C:\PDFCreator-Temp\
2.) Redémarrez l'ordinateur !
3.) Lancez PDFCreator, choisissez Options/Répertoires puis tapez %PTPath%<Redmon_User>. Vous devriez voir : C:\PDFCreator-Temp\'Preview REDMON_USER'\
