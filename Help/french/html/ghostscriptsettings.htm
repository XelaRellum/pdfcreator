Paramètres Ghostscript

Les réglages Ghostscript permettent de régler les propriétés de la version de Ghostscript utilisée.

Version Ghostscript : Liste les versions de Ghostscript installées sur votre machine. La valeur par défaut est la version interne de Ghostscript fournie avec PDFCreator. Changez ce paramètre seulement si vous voulez utiliser une version préinstallée de Ghostscript.

Paramètres Ghostscript additionnels : Avec cette option, ajoutez votre propre liste de paramètres Ghostscript. Utilisez le symbole | pour séparer les paramètres. Cette option est vide par défaut.
Exemple :
AdditionalGhostscriptParameters=-dTextAlphaBits=4|-dGraphicsAlphaBits=4|-dDOINTERPOLATE

Chemin des ressources Ghostscript additionnelles : Cette option permet à l'utilisateur de définir un chemin supplémentaire où Ghostscript peut trouver des ressources. Par exemple, vous pouvez ajouter un chemin vers des polices supplémentaires.

Utiliser les polices Windows : Ce paramètre permet à l'utilisateur d'ajouter le chemin des polices Windows aux chemins des polices Ghostscript. L'option est activée par défaut.
Valeurs :
0 - off
1 - on (Default)
