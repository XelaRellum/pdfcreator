Version 1.1.0 (14 Novembre 2010)

    Améliorations :
  • Utilisation de Ghostscript 9.0 désormais.
  • Support du cryptage AES 128 bits pour les fichiers PDF.
  • Installation améliorée : les anciennes versions de PDFCreator n'ont pas à être désinstallées au préalable.
    Correction de bug :
  • Correction de quelques bugs mineurs.
    Problèmes connus :
  • Impossible de créer des liens hypertextes (Créer des liens hypertextes pour MS Word est possible par la combinaison de PDF-T-Maker et PDFCreator).

Version 1.0 (28 Mai 2010)

    Améliorations :
  • Ajout de la possibilité d'ajuster manuellement le facteur de compression couleur/gris pour optimiser le compromis taille/qualité des fichiers PDF.
  • Amélioration de l'option d'impression après sauvegarde (impression couleur, réglage de la résolution maximale).
  • Amélioration de la fonction de signature (l'affichage de la signature dans la page peut être librement défini).
  • PDFCreator peut assurer la signature et le chiffrage en une seule opération.
  • Amélioration de l'installation : le pilote d'impression 64 bits peut être installé sur un système 32 bits.
  • Amélioration des boîtes de dialogue pour le mot de passe de certificat.
  • Ajout du choix de polices pour les menus des options.
  • Amélioration de la classe PDFCreator (recherche d'une imprimante existante dans cPrintPrinterTestpage, recherche d'un fichier existant dans cPrintfile)
    Correction de bugs :
  • Correction du plantage d'un menu des options
  • Correction du bug "481 - invalid picture"
  • Correction d'un bug de lecture des options sur les systèmes Win9xMe
  • Correction d'un bug si l'utilisateur utilise un format de calendrier Thai
  • Correction d'un bug en cas d'utilisation de versions différentes de Ghostscript
  • Corrections de bugs mineurs
  • Impossible de créer des liens hypertextes (Créer des liens hypertextes pour MS Word est possible par la combinaison de PDF-T-Maker et PDFCreator).

Version 0.8.0 (2 avril 2004)

    Améliorations :
  • Il est maintenant possible de nous faire un don. (http://www.pdfforge.org)
  • Sécurité ! PDFCreator peut au choix utiliser les options de sécurité de AFPL Ghostscript 8.14 et au-delà, ou bien utiliser le programme 'pdfenc' de Steven Lee.
  • Meilleure gestion des pilotes d'imprimante.
  • 2 nouvelles variables : OptionsEnable et OptionsVisible (une simple protection pour sauvegarder les options - utile aux administrateurs).
  • Il faut ajuster ces variables manuellement dans le fichier PDFCreator.ini pour sauvegarder les options.
  • Clients Email : Testé successivement avec MS Outlook, Outlook Express, Thunderbird, Eudora, Pegasus Mail (WsendTo.exe), The Bat!, Foxmail, Sylpheed-Claws, IncrediMail. Problems with: Ak-Mail, GeMail, Postguard, Vivian Mail.
  • Ajout des propriétés 'Mots clés' et 'Sujet' des documents PDF.
  • Vous pouvez ajouter des fichiers imprimables (aussi avec 'Glisser/Déplacer') et au travers des menus contextuels de l'explorateur Windows.
  • Ajout du paramètre de ligne de commande PF: Convertit un fichier imprimable en pdf. Exemple: PDFCreator.exe -PF"c:\autoexec.bat".
  • Recherche des mises à jour depuis le menu de PDFCreator.
    Correction de bugs :
  • Problème avec la fonction e-mail corrigé.
  • Problème avec la fonction de fusion corrigé.
  • Correction de quelques bugs mineurs.
    Problèmes connus :
  • Ne fonctionne pas en session à distance sur terminalservers (Merci d'être patients, nous travaillons d'arrache-pied sur ce problème).
  • Pas de liens hypertextes possibles.
