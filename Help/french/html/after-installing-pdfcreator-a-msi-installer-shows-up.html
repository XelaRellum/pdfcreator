Aide PDFCreator
 
 

Après l'installation de PDFCreator, une fenêtre de l'installateur MSI apparaît.

Cette fenêtre apparaît à cause d'une mauvaise installation de MSI. Pour résoudre ce problème :

  1. exécutez l'installation proposée ou
  2. utilisez l'utilitaire gratuit Windows Installer CleanUp de Microsoft qui réparera l'installation.
