Compression

Ce menu ajuste les paramètres de compression utilisés pour la création du fichier PDF par PDFCreator.

Comprimer les objets Texte : Lorsque l'option est utilisée, PDFCreator compacte le texte dans le document pour réduire la taille du fichier.

Compression des images : Comme pour les objets Texte, les images peuvent être compactées pour réduire la taille du PDF. Trois types d'images peuvent être compressées : images couleur, images en niveaux de gris et images monochromes. Voici les options de compression des images :

  • Type de compression : Règle le type de compression utilisé pour réduire la taille des images. Plusieurs types de compression peuvent être utilisés, y compris JPEG et ZIP. Pour la compression JPEG, vous pouvez aussi ajuster le niveau de compression. Un haut niveau de compression réduit la taille du fichier, mais réduit aussi la qualité de l'image. Vous pouvez ajuster manuellement le taux de compression afin d'optimiser le ratio taille/qualité.
  • Rééchantillonner : Définit comment l'image est recalculée lors de la compression. Le rééchantillonnage réduit le nombre de pixels de l'image, donc le nombre de pixels utilisés est réduit également. Le rééchantillonnage réduit la taille de tout fichier image.
  • Résolution : Fixe le nombre de ppi, ou points par pouce, que les images auront une fois compressées dans le fichier PDF.
