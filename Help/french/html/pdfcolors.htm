Paramètres PDF : Couleurs

Le menu couleurs permet à l'utilisateur d'adapter le modèle de couleurs et les autres paramètres de couleur utilisés lorsque PDFCreator crée un PDF.

Modèle de couleur : Le modèle de couleur est un système qui définit les couleurs par des valeurs numériques. PDFCreator peut utiliser trois modèles de couleur différents : RVB (Rouge, Vert, Bleu), CMJN (Cyan, Magenta, Jaune, Noir), et Niveaux de gris.

Convertit les images CMJN en RVB : Cette option convertit les images du modèle CMJN en RVB.

Préserver les options de surimpresssion : Cette option permet de conserver les paramètres de surimpression dans tout le PDF.

Préserver les fonctions de transfert : Cette option permet de conserver les informations de transfert dans tout le document PDF.

Préserver les demi-tons : Cette option préserve les informations sur les demi-tons dans tout le document PDF.
