Aide PDFCreator
 

Fusionner des documents

Fusionner des documents est une fonctionnalité très utile, qui vous permet de prendre deux documents distincts et d'en faire un seul document PDF. Fusionner plusieurs documents est aussi facile qu'imprimer plusieurs documents.

1. Ouvrez les documents que vous souhaitez fusionner.

2. Imprimez le premier document comme vous le feriez sur une imprimante standard.

3. Lorsque la fenêtre de dialogue pour l'impression s'affiche, sélectionnez l'imprimante PDFCreator à la place de l'imprimante par défaut. Si l'imprimante PDFCreator est déjà votre imprimante par défaut, ne changez rien. Ensuite, cliquez sur OK.

4. Une autre fenêtre apparaît alors, et vous demande de saisir plusieurs informations pour créer votre PDF. Cliquez alors sur le bouton "Mettre en file d'attente". Le moniteur d'impression de PDFCreator apparaît et le fichier se place dans la file d'attente.

5. Ouvrez le second document et imprimez-le en utilisant l'imprimante PDFCreator.

6. Le moniteur d'impression de PDFCreator fait apparaître maintenant les deux documents dans la file d'attente. Sélectionnez les deux documents et allez dans le menu "Document" puis cliquez sur "Fusionner". Les deux documents n'en forment plus qu'un.

7. Allez maintenant dans le menu "Document" et cliquez sur "Imprimer". PDFCreator vous demandera plusieurs informations ainsi que l'endroit où enregistrer le PDF, ou bien si vous souhaitez l'envoyer par email.

8. Votre document PDF sera créé et ouvert dans votre logiciel de lecture des PDF par défaut, ou le cas échéant il sera attaché à un nouveau message dans votre client de messagerie par défaut.
