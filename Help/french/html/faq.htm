Aide PDFCreator
 
 

Questions fréquemment posées

Q: Quelle est la différence entre la version AFPL et la version GPL de PDFCreator?

R: La principale différence réside dans la version de Ghostscript utilisée. La version AFPL est plus récente que la version GPL, mais la license est plus restrictive. Si vous souhaitez simplement utiliser PDFCreator, alors les deux versions peuvent convenir, et la version AFPL est préférable car bien meilleure. Si vous avez besoin d'un aspect de license pris en compte par GPL et non par AFPL, alors téléchargez la version GPL.

Q: Après l'installation, j'obtiens un message d'erreur "Exception EInOutError in module is-XXXXX.exe at 0000631C. File not found". Que faire ?

R: C'est un problème lié aux installations antérieures réalisées à l'aide de InstallShield. Dans la plupart des cas, vous pouvez sans risque supprimer de votre répertoire Windows le fichier dont le nom est donné. Si vous n'êtes pas sûr de son importance, déplacez-le dans un répertoire de votre choix pour le sauvegarder.

Q: Je ne peux ni faire une recherche, ni Copier/Coller dans le texte de mon fichier PDF. Que faire ?

R: C'est un problème dû à un mauvais pilote d'imprimante installé par PDFCreator 0.7.1. Désinstallez PDFCreator, redémarrez l'ordinateur et installez la dernière version.

R: J'ai installé PDFCreator mais il ne fonctionne pas, que faire ?

R: Vérifiez si un pilote d'impression est installé. Sinon, essayez de désinstaller PDFCreator. Redémarrez l'ordinateur et installez PDFCreator à nouveau. Redémarrez l'ordinateur à nouveau et vérifiez si une umprimante est installée. Sinon, examinez le fichier SetupLog.txt dans le répertoire de PDFCreator. Si une erreur y est répertoriée, communiquez-la nous dans les forums de SourceForge.net.

Q: L'imprimante n'est pas compatible avec une règle d'utilisation de votre ordinateur qui bloque les pilotes NT 4.0

R: Le message d'erreur est dû à une règle commune qui interdit l'installation d'imprimantes utilisant des pilotes en mode kernel. Les pilotes en mode kernel ont accès à la mémoire système, et les drivers mal conçus peuvent créer des blocages du sytème. Pour permettre l'installation de ce type de pilotes, suivez les étapes suivantes :

  1. Cliquez sur Démarrer, puis sur Exécuter...
  2. Saisissez gpedit.msc et pressez ENTER
  3. Allez sur Computer Configuration/Administrative Templates/Printers et réglez l'option "Disallow installation of printers using kernel-mode drivers" sur "Disabled"

Q: Pourquoi le texte noir ressort gris ?

A: Le modèle de couleurs CMJN fait apparaître le texte noir en gris, bien qu'il y ait un niveau de noir de 100%. Pour résoudre ce problème, choisissez le modèle de couleurs RVB dans les réglages de PDFCreator.

Q: Est-ce que PDFCreator fonctionne avec le programme XYZ ?

A: PDFCreator fonctionne avec tous les programmes capables d'imprimer sous Windows. Le résultat aura l'aspect qu'il aurait en cas d'impression normale, ce qui est généralement conforme à ce qui apparaît sur l'écran.
