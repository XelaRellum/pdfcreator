The degree symbol overlaps the following character

This happens when an application uses a standard font (i.e. Arial). When printing this, the font will be replced by an internal printer font which may have slightly different sizes of the glyphs. This can be fixed by embedding the fonts in the print job:

In the printer settings there is a tree of settings below the advanced settings. Under Graphics -> TrueType fonts is the setting "Replace with device font". Setting this to "Load as Softfont" will fix the issue.
