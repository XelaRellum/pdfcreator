Aide PDFCreator
 
 

Options de PDFCreator

Depuis la version 0.8.1 RC8, PDFCreator enregistre tous les réglages dans le registre. Si vous choisissez l'installation "Standard", PDFCreator enregistre les réglages à cet emplacement du registre : HKCU\Software\PDFCreator

Si votre système d'exploitation est Windows 2000 ou une version ultérieure, le panneau de réglages stocke quelques paramètres pour l'utilisateur par défaut à l'emplacement suivant du registre : HKU\.DEFAULT\Software\PDFCreator

En cas d'installation de type "Serveur", les réglages sont enregistrés à l'endroit suivant du registre : HKLM\Software\PDFCreator

Alors que la localisation par défaut des réglages est le registre, il est également possible de stocker les réglages selon l'ancien système des fichiers .ini. Dans ce cas PDFCreator stocke tous les réglages dans des fichiers .ini. On peut aussi ajouter un fichier .ini externe comme fichier par défaut. Pour plus d'information utiliser le paramètre /? du programme d'installation.
Attention : Des problèmes peuvent survenir avec des profils basés sur serveur lorsque le dossier Application Data est sur un lecteur réseau. Dans ce cas les installations sur serveur ne peuvent utiliser l'ancien système .ini.

Si vous choisissez l'installation "Standard" et utilisez l'enregistrement dans les fichiers .ini, les réglages sont enregistrés dans le dossier Application Data du répertoire de l'utilisateur:
Exemple : C:\Documents and Settings\[user profile]\Application Data\PDFCreator

Si votre système d'exploitation est Windows 2000 ou une version ultérieure, le panneau de réglages enregistre quelques réglages pour l'utilisateur par défaut dans le répertoire suivant :
Exemple : C:\Documents and Settings\Default User\Application Data\PDFCreator

Dans une installation de PDFCreator de type "Serveur", les réglages sont enregistrés dans le répetoire suivant :
C:\Documents and Settings\All User\Application Data\PDFCreator

Procédure pour lire les paramètres

Sous Windows 2000 ou une version ultérieure, PDFCreator lit les paramètres de l'utilisateur par défaut en priorité. S'il n'existe pas de paramaètres à cet endroit, PDFCreator utilisera des paramètres externes. Ensuite, PDFCreator lit les paramètres de l'utilisateur courant. Seuls les réglages de l'utilisateur courant trouvés par PDFCreator remplaceront les réglages de l'utilisateur par défaut. Enfin, PDFCreator recherche les réglages de la machine dans
HKLM\Software\PDFCreator
ou
C:\Documents and Settings\All User\Application Data\PDFCreator.
Seuls les réglages de la machine trouvés par PDFCreator remplaceront les réglages de l'utilisateur. Grâce à se fonctionnement, les administrateurs peuvent définir des réglages pour tous les utilisateurs.

Procedure d'enregistrement des réglages

Si vous utilisez l'installation "Standard" de PDFCreator, les réglages sont stockés dans l'un des emplacements suivants :
HKCU\Software\PDFCreator
ou
C:\Documents and Settings\[user profile]\Application Data\PDFCreator

Si vous choisissez l'installation "Serveur" de PDFCreator, les réglages sont enregistrés dans l'un des emplacements suivants :
HKLM\Software\PDFCreator
ou
C:\Documents and Settings\All User\Application Data\PDFCreator

Astuce :  Si vous voulez utiliser des réglages différents de ceux par défaut, utilisez les options /REGFile= ou /INIFile= du programme d'installation. Pour plus d'informations sur les options du programme d'installation, utilisez l'option /?.
