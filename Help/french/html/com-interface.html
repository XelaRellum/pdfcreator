COM Interface

Depuis la version 0.8.1 RC9, PDFCreator propose une interface COM. Il est donc possible de contrôler PDFCreator en utilisant la ligne de commande de toute autre application.
Presque toute les fonctionnalités sont accessibles depuis l'interface COM. L'interface COM comprend des classes publiques, des variables publiques, des méthodes publiqueset des écènements publics.
Il n'y pas d'UDTs utilisés donc la compatibilité est assurée en cas de mise à jour d'UDTs.
Les noms sont "auto-documentés", donc aucune explication supplémentaire n'est nécessaire.

Si l'interface COM doit être testée en mode serveur, tous les réglages doivent être enregistrés avec la fonction cSaveOptions sinon les changements n'auront aucun effet.

Dans le sous-répertoire "COM" (situé dans le répertoire PDFCreator), on trouvera plusieurs exemples dans différents langages de programmation pour illustrer ces procédures.
