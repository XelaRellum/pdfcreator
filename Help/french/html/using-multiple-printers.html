Aide PDFCreator
 

Utiliser des imprimantes multiples

PDFCreator vous permet d'utiliser plusieurs imprimantes. Dans la fenêtre principale, vous pouvez accéder au paramétrage des imprimantes en suivant Imprimante -> Imprimantes.

Ajouter/Supprimer une imprimanted

Vous pouvez utiliser PDFCreator pour ajouter de nouvelles imprimantes. Ces imprimantes ne sont pas des imprimantes ordinaires, mais sont destinées à l'utilisation de PDFCreator. Tapez simplement le nom de l'imprimante et cliquez sur "Ajouter une imprimante". Après avoir sélectionné une imprimante, vous pouvez la supprimer en cliquant sur "Supprimer une imprimante". Cela ajouter/supprime des imprimantes Windows.

Note : Vous devez disposer des privilèges Administrateur pour ajouter ou supprimer des imprimantes. Sous Windows Vista et systèmes ultérieurs, vous devez également exécuter PDFCreator en tant qu'aministrateur.

Lier une imprimante à un profil

Vous pouvez relier chaque imprimante à un profil de paramètres. Vous pouvez disposer d'une imprimante pour les tâches qui nécessitent d'enregistrer vos travaux en PDF dans un dossier particulier, et une autre imprimante pour les tâches standard.

Autres possiblités :

  • une imprimante pour les images JPEG
  • PDF haute résolution destinés à l'impression
  • PDF basse résolution pour les eMails
  • une imprimante PDF avec un auteur, des paramètres de sécurité prédéfinis, etc.

Pour apprendre à utiliser les profils, référez-vous à la section "Utilisation des profils".
