Aide PDFCreator
 
 

Créer un fichier de traduction

Pour créer un fichier de traduction, ouvrez l'outil de traduction (Translation Tool) depuis le dossier PDFCreator. Une fenêtre apparaît avec quatre colonnes de paramètres. Voici une liste de ces paramètres et de leur fonction :

  • Section : Cette colonne indique à l'utilisateur à quelle partie du logiciel a trait le texte en question. Cette colonne est purement informative et ne peut être modifiée.
  • Key : Cette colonne indique au programme où utiliser le texte en question. Cette colonne est purement informative et ne peut être modifiée.
  • Template Text : Cette colonne contient le texte que vous allez traduire dans une autre langue. Cette colonne ne peut être modifiée.
  • Translated Text : Cette colonne contient la traduction que vous allez faire. Lorsque vous cliquez sur une ligne, une boîte de dialogue apparaît où vous pouvez saisir votre texte.

Si vous souhaitez traduire depuis une autre langue que l'anglais, allez dans le menu File et cliquez sur Open Template File. Dans la fenêtre suivante, choisissez la langue à partir de laquelle vous voulez faire une traduction.

Il y a 330 phrases différentes à traduire dans PDFCreator. Une fois l'ensemble des termes traduits, enregistrez le fichier dans le répertoire languages du répertoire d'installation de PDFCreator. Le fichier doit être enregistré en tant que fichier .ini.
