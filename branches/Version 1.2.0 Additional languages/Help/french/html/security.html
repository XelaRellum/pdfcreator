Aide PDFCreator
 
 

Sécurité

Le menu Sécurité permet d'ajuster les paramètres de protection pour crypter les documents PDF et de désactiver plusieurs fonctionnalités pour une plus grande sécurité du document.

Utiliser les options de sécurité : Active ou désactive les options de sécurité de PDFCreator.

Cryptage : PDFCreator peut utiliser deux systèmes de cryptage. Le premier est inclus dans Ghostscript 8.14 et les versions ultérieures. Le second est le système pdfenc de PDFCreator. Les deux proposent une sécurité avancée avec protection par mot de passe et cryptage jusqu'à 128 bits.

Niveau de cryptage : Le cryptage permet d'empêcher la copie non autorisée d'informations contenues dans le PDF. PDFCreator peut crypter selon trois niveaux : Bas (40 bits),  Haut (128 bits) et très Haut (AES 128 bits). Le cryptage AES est disponible seule si le fichier pdfforge.dll existe. Cette Dll est installée avec PDFCreator si .Net 2.0 est installé.

Mots de passe : Pour la sécurité, vous pouvez exiger un mot de passe pour visionner le document (mot de passe utilisateur) et pour modifier les permissions et les mots de passe (mot de passe propriétaire). Losque l'option est sélectionnée, PDFCreator demandera les mots de passe utilisateur et/ou propriétaire au moment de la création du PDF.

Interdire à l'utilisateur : Cette section permet au propriétaire du PDF de déterminer ce que les utilisateurs du document peuvent faire. Le propriétaire peut interdire à l'utilisateur d'imprimer ou copier textes et images, de modifier le document et de faire des changements ou des commentaires.

Permissions étendues : Ces permissions concernent seulement le cryptage 128 bits. Le propriétaire peut autoriser les utilisateurs à imprimer seulement en basse résolution, remplir des formulaires, la lecture à l'écran et changer de langue.
