Colors

The colors menu allows the user to modify the color model and other color settings that are used when PDFCreator makes a PDF.

Color Model: The color model is a system that specifies colors in a numerical value. PDFCreator can use three different color models: RGB (Red, Green, Blue), CMYK (Cyan, Magenta, Yellow, Black), and Grayscale.

Convert CMYK images to RGB: When checked, PDFCreator converts any images in the CMYK color model into the RGB color model.

Preserve Overprint Settings: When checked, saves the overprint settings along with the typical data in the PDF file.

Preserve Transfer Functions: When checked, saves the transfer information along with the typical data in the PDF file.

Preserve Halftone Information: When checked, saves the color halftone information along with the typical data in the PDF file.
