Hidden settings

PDFCreator includes several hidden settings that are used for testing purposes. These default storage area for these features is in the registry.

ClientComputerResolveIPAddress: In special cases is it possible that the token <ClientComputer> contains an IP-address instead a hostname. If you enable this feature, PDFCreator will try to resolve the IP-address into a hostname.
Properties:
0 - Off (Default)
1 - On

DisableEmail: This setting deactivates the email button in PDFCreator.
Properties:
0 - Email button is enabled (Default)
1 - Email button is disabled

DisableUpdate: This setting deactivates the update check. PDFCreator will not perform automatic checks for new versions. The Check now button in the settings and the menu item Check for Updates in the help menu are disabled.
Properties:
0 - Update is enabled (Default)
1 - Update is disabled


PDFCompressionColorCompression*, PDFCompressionGreyCompression*: With help of these parametres it is possible to set individual compression factors for each compression degree so the compression of colour and greyscale pictures can be controlled much better. All values must be greater than 0.
Default values:
PDFCompressionColorCompressionJPEGMaximumFactor=2
PDFCompressionColorCompressionJPEGHighFactor=0.9
PDFCompressionColorCompressionJPEGMediumFactor=0.5
PDFCompressionColorCompressionJPEGLowFactor=0.25
PDFCompressionColorCompressionJPEGMinimumFactor=0.1
PDFCompressionGreyCompressionJPEGMaximumFactor=2
PDFCompressionGreyCompressionJPEGHighFactor=0.9
PDFCompressionGreyCompressionJPEGMediumFactor=0.5
PDFCompressionGreyCompressionJPEGLowFactor=0.25
PDFCompressionGreyCompressionJPEGMinimumFactor=0.1

PDFOwnerPasswordString: This setting is used when security is enabled for PDF documents. When used, PDFCreator will automatically put in the password supplied in the PDFCreator.ini file for the owner password. WARNING: Because the value is not encrypted, you use this setting at your own risk.

PDFUserPasswordString: This setting is used when security is enabled for PDF documents. When used, PDFCreator will automatically put in the password supplied in the PDFCreator.ini file for the user password. WARNING: Because the value is not encrypted, you use this setting at your own risk.

RemoveAllKnownFileExtensions: This setting enables PDFCreator to remove all known file extensions beginning, from the right, until there is no known file extension. For example, the internal title of postscript file is 'Pic.jpg.-12.31.2005.pcx'. The title then becomes 'Pic.jpg.-12.31.2005'. This setting improves the save process.
Properties:
0 - off
1 - on (Default)

StandardCreationdate: This setting enables the user to place a standard creation date on all documents created by PDFCreator.

StandardDateformat: This setting enables the user to change the date format that PDFCreator uses to make its creation dates. Not all date formats are allowed. Please check the settings for your system for correct date formats.

StandardKeywords: This setting enables the user to place a standard set of keywords in all documents created by PDFCreator.

StandardModifydate: This setting enables the user to place a standard last modified date on all documents created by PDFCreator.

StandardSubject: This setting enables the user to place a standard subject on all documents created by PDFCreator.

StandardTitle: This setting enables the user to place a standard title on all documents created by PDFCreator.
