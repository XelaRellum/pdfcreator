PDFCreator Hilfe
 
 

Aktualisierung des Programmes

Es ist sehr einfach zu überprüfen, ob Aktualisierungen für PDFCreator vorliegen. Starten Sie dazu das Programm PDFCreator aus dem Startmenü, klicken Sie dann auf den Menüpunkt ? und dann auf Aktualisierungen suchen. Bitte stellen Sie vorher sicher, das Sie mit dem Internet verbunden sind.
