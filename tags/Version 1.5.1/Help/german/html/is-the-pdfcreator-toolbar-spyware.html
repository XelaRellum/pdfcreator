Ist die PDFCreator Toolbar Spyware?

Ziel der Toolbar ist es dem Benutzer eine erweiterte Funktionalität zu bieten. Somit soll es nicht nur möglich sein Suchanfragen zu stellen, sondern auch PDF-Dateien aus Webseiten direkt zu erstellen. Dabei versendet die Toolbar keine Daten ohne das Wissen des Benutzers, wie es bei Spyware üblich ist. Es wird sogar durch eine mögliche Aktualisierung darauf geachtet, dass die Toolbar den aktuellen Sicherheitsstandards entspricht.

Das Unternehmen, das für PDFCreator die Toolbar erstellt hat, testet die Toolbar sowie PDFCreator mit Produkten bekannter AntiViren-Software Herstellern.

Wenn jedoch Bedenken bezüglich der Toolbar weiterhin bestehen, kann diese im Setup unter PDFCreator Browser AddOn abgewählt oder nachträglich jederzeit deinstalliert werden.
