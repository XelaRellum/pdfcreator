Eine Sprachdatei erstellen

Um eine Sprachdatei zu erstellen, öffnen Sie bitte das Übersetzungswerkzeug TransTool aus dem Programmverzeichnis von PDFCreator. Ein Fenster mit vier Spalten erscheint. Die Spalten haben folgende Bedeutung:

  • Section: Diese Spalte sagte, aus welchem Bereich der Anwedung der Text kommt. Diese Spalte ist sehr wichtig und kann nicht geändert werden.
  • Key: Diese Spalte legt fest, an welcher Stelle der Text angezeigt wird. Diese Spalte ist sehr wichtig und kann nicht geändert werden.
  • Template Text: Diese Zeile enthält den zu übersetzenden Text. Diese Spalte kann nicht geändert werden.
  • Translated Text: In diese Spalte kommt der übersetzte Text. Wenn Sie in eine Zeile klicken erscheint ein Textfeld, in das der Text eingegeben werden kann.

Wenn sie aus einer anderen Sprache als Englisch übersetzen möchten, dann gehen Sie ins File Menü und wählen Open Template File. Wählen Sie im nächsten Fenster die Sprache aus, die Sie als Vorlage benutzen möchten.

Es gibt über 330 Schlüssel in PDFCreator zu übersetzen. Wenn diese alle übersetzt sind, dann speichern Sie die fertige Datei im Ordner für Sprachdateien von PDFCreator. Die Datei sollte als .ini Datei gespeichert werden.
