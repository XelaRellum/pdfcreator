BMP Einstellungen

In diesem Menü können die Einstellungen für das Erstellen von BMP-Bildern vorgenommen werden. Das BMP Format sollte nur zum erstellen von einseitigen Dokumenten genutzt werden.

Auflösung: Legt die dpi (dots per inch = Punkte pro Zoll) des Bildes. Je höher die dpi, desto detailreicher und größer wird das Bild. Der Standardwert ist 150 dpi.

Farben: Legt die Anzahl Farben im BMP Bild fest. Die beste und größte Einstellung ist 24 Bit (16,7 Mio) Farben.
