PDFCreator Hilfe
 
 

Installation einer Sprachdatei

Auf der Webseite von PDFCreator befindet sich eine große Anzahl von Sprachdateien. Falls benötigt, laden Sie sich die erforderliche Sprachdatei herunter und kopieren Sie diese Datei in das Verzeichnis Languages im Programmverzeichnis. Nach dem nächsten Programmstart steht diese Sprache im Menüpunkt Sprachen zur Verfügung.
