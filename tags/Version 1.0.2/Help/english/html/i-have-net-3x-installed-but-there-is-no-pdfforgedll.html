I have .Net 3.x installed, but there is no pdfforge.dll

This is a bit tricky as .Net 2 and above do not include .Net 1.1. Microsoft does not even preinstall .Net 1.1 von Vista and Windows 7 anymore. Please install the .Net 1.1 Redistributable and then reinstall PDFCreator. This will bring you the pdfforge.dll.
