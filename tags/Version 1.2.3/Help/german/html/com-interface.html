PDFCreator Hilfe
 
 

COM-Schnittstelle

Seit der Version 0.8.1 RC9 verfügt der PDFCreator über eine COM-Schnittstelle. Damit wird es möglich, den PDFCreator von der Kommandozeile oder aus anderen Anwendungen zu steuern.
Fast alle Möglichkeiten des Programms sind auch über die COM-Schnittstelle verfügbar. Die COM-Schnittstelle verfügt über öffentliche Klassen, öffentliche Eigenschaften, öffentliche Methoden und öffentliche Ereignisse.
UDTs wurden nicht verwendet, damit die Kompatibilität nicht bei jeder Erweiterung eines UDTs gebrochen wird.
Es wurden "sprechende" Namen verwendet, so dass die Funktion aus dem Namen hervorgeht und eine weitere Beschreibung nicht nötig ist.

Sollte die COM-Schnittstelle im Servermodus getestet werden, müssen die Einstellungen immer mit der Function cSaveOptions gespeichert werden. Ansonsten haben Änderungen keinen Effekt.

Im Unterverzeichnis COM sind verschiedene Beispiele für unterschiedliche Programmiersprachen vorhanden, die die Vorgehensweise demonstrieren sollen.
