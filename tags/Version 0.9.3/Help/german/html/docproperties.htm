PDFCreator Hilfe
 
 

Dokumenteinstellungen

Die Dokumenteinstellungen erlauben dem Benutzer allgemeine Einstellungen für alle von PDFCreator erstellten Dokumente festzulegen.

Standardautor festlegen: Wenn dieses Einstellung aktiviert wurde, wird ein Standardautor zu jeder erstellen PDF-Datei hinzugefügt. Es können auch zusätzliche Felder zum Namen, wie Computername, DateTime, Title, und Username, ClientComputer und die Redmon-Variablen hinzugefügt werden. Weitere Hinweise dazu siehe Automatisches Speichern.

Benutze aktuelles Datum als 'Erstellungsdatum': Wenn dieses Feld angekreuzt ist, wird die aktuelle Zeit anstelle der Zeit, zu der das Dokument ursprünglich erstellt wurde, benutzt.

Eine Seite je Datei: Die Einstellung erlaubt Benutzern eine neue Datei für jede Seite des Originals zu erstellen. Aufgrund von Einschränkungen in Ghostscript funktioniert das nicht mit PDF und EPS Dateien.

Stempelzeichenkette: Wenn diese Einstellung gesetzt ist, legt PDFCreator einen Stempel über das Dokument. Sie können die Schriftart, Farbe, Größe und Umrandung des Stempels mit den folgenden Einstellungen festlegen. Diese Einstellung ist noch sehr experimentell. Bitte kontrollieren Sie die erstellten Dokumente.

Schriftartfarbe: Legt die Farbe für den Stempeltext fest. Verwenden Sie dafür HTML-Farben.
Beispiel für Rot: StampFontColor=#FF0000

Schriftumrandung verwenden: Diese Einstellung legt fest, ob eine Umrandung für den Stempeltext benutzt wird.
Eigenschaften:
0 - Aus (Standard)
1 - Ein

Dicke der Schriftumrandung: Legt die Breite der Umrandung für den Stempeltext fest.

Feste Seitegröße verwenden: Diese Einstellung kann verwendet werden, um eine Standardpapiergröße vorzugeben.

Benutzerdefinierte Seitengröße verwenden: Diese beiden Einstellungen legen eine fest vorgegebene Breite und Höhe für das zu verwendene Papierformat fest. Die Einheit ist 1/72 eines Zolls und kann als Alternative für die Einstellung Feste Seitegröße verwenden verwendet werden, aber nicht in Kombination damit.
