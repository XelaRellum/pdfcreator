PCX-Einstellungen

Dieser Menüpunkt erlaubt Einstellungen für das PCX Bildformat vorzunehmen. Das PCX-Format sollte nur bei einzelnen Seiten verwendet werden.

Auflösung: Einstellen der DPI eines Bildes. Je höher die DPI Zahl ist, um so detailierter, aber auch um so größer wird das Bild. Der Standardwert ist 150.

Farben: Hier kann die Anzahl der Farben ausgewählt werden. Die beste und höchstmögliche Auswahl sind 32 Bit Farben (CMYK).
