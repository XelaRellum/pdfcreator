PDF Einstellungen: Komprimierung

In diesem Menü können verschiedene Einstellungen vorgenommen werden, um eingebettete Objekte in PDF-Dateien komprimiert zu speichern.

Komprmiere Textobjekte: Wenn diese Option ausgwählt wird, wird der enthaltene Text komprimiert.

Bildkomprimierung: Genauso wie der Text, können auch Bilder komprimiert werden, um die Dateigröße zu verringern. Es können Einstellungen für 3 verschiedene Bildarten vorgenommen werden: Farbbilder, Graustufenbilder und Schwarz-Weiß-Bilder. Es folgen die unterschiedlichen Einstellungsmöglichkeiten für diese Bildarten:

  • Kompressionsart: Legt die verwendete Kompressionsart fest. Sie können die Komprimierung autumatisch vornehmen lassen oder selber das Komprimierungsverfahren festlegen (JPEG, Zip, LZW).
  • Resample: Wählen Sie Resample, wird die Auflösung eines Bildes entsprechend der gewählten Auflösung geändert. Dies kann die Dateigröße erheblich verringern. Zusätzlich können Sie festlegen, wie diese Umrechnung zu erfolgen hat, um die bestmögliche Qualität zu erhalten.
  • Auflösung: Legt die DPI (dots per inch = Punkte pro Zoll) eines komprimierten Bildes fest, für das die Option Resample gewählt wurde.
