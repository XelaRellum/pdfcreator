Errreur d'installation : PDFCreator est déjà installé

Cela arrive parfois que l'installateur n'installe pas PDFCreator, car il détecte une installation antérieure. Cela peut être dû à une désintallation interrompue ou une installation défectueuse (honte sur nous dans ce cas). Vous obtenez alors une erreur "Ce programme est déjà installé.
Pour une nouvelle installation, désinstallez le programme en premier lieu".

Il arrive parfois aussi que certaines personnes exminent le code source de notre installateur et en copie des parties qui les intéressent, ce qui est parfaitement normal et correspond tout à fait à ce pour quoi l'open source existe. Mais ils conservent un réglage qui permet à PDFCreator de s'identifier lui même. Un conflit apparaît alors.

Pour résoudre ce problème, un paramètre d'installation nommé /ForceInstall permet d'ignorer ce réglage et devrait résoudre tous les problèmes de ce type.

Vous pouvez aussi lancer l'installatin à partir de la ligne de commande (Start->Execute->" /ForceInstall") ou créer un raccourci vers l'installateur et ajouter le paramètre /ForceInstall à ses propriétés.
