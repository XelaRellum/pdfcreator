PDFCreator Hilfe
 
 

Nach der Installation des PDFCreators wird ein Installationsdialog des MSI Installers gezeigt

Der Dialog wird auf Grund einer fehlerhaften MSI Installations angezeigt. Um dieses Problem zu lösen kann man

  1. die Installation des jeweilige Programmes zu Ende führen zu lassen oder
  2. mit dem kostenfreien Programm Windows Installer CleanUp von Microsoft die fehlerhafte Installation entfernen.
