.Net 3.x est installé, mais il n'y a pas pdfforge.dll

C'est un peu délicat car .Net 2 et les versions suivantes n'incluent pas .Net 1.1. Microsoft n'installe plus non plus .Net 1.1 depuis Vista et Windows 7. Installez .Net 1.1 Redistributable et réinstallez ensuite PDFCreator. Cela vous permettra de récupérer pdfforge.dll.
