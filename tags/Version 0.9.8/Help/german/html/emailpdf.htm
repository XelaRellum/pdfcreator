PDFCreator Hilfe
 
 

Eine PDF-Datei per Email verschicken

Eine PDF-Datei zu verschicken, geht genauso leicht wie das normale Erstellen.

1. Öffnen Sie das zu konvertierende Dokument in Ihrer gewohnten Anwedung.

2. Drucken Sie die Datei, als würden Sie sie auf einem normalen Drucker ausdrucken.

3. Wenn der Drucken-Dialog erscheint, wählen Sie den Drucker PDFCreator aus. Wenn PDFCreator bereits der Standarddrucker ist, muss nichts geändert werden.

4. Ein weiteres Fenster erscheint, wo Sie mehrere Informationen zur PDF-Datei angeben können. Drücken Sie auf die Schaltfläche 'Email' um die PDF-Datei jetzt zu erstellen oder auf 'Warten', um das zu einem späteren Zeitpunkt zu tun.

    Hinweis: Hier können Sie auch das Ausgabeformat für das Dokument bestimmen. Sie haben die Wahl zwischen PDF, PNG, JPEG, BMP, PCX, TIFF, PS und EPS.

5. Wenn sie 'Email' gewählt haben wird die PDF-Datei erstellt und in der Standardanwendung für Mails geöffnet.
