Directories Properties

The directories properties window allows the user to change the directories where PDFCreator accesses or stores information and files. By default, PDFCreator sets these directories for you, but you may change them to fit your system better. It is not recommended to change these directories.

Temporary Files: The directory where PDFCreator temporally creates files when making documents. It is possible to use 'PDFCreator Tokens' and environment variables in this setting, but please be carefull with this option. Not every token and environment variable made sense for this setting. It is recommend that you use only the following tokens: <Temp>, <Redmon_User>, <Redmon_Machine>, <Clientcomputername> and <Computername>. If you use environment variables, please use only system environment variables and reboot your computer after you set a system environment variable.

Example:
1.) Set the system environment variable PTPath: PTPath=C:\PDFCreator-Temp\
2.) Reboot the computer!
3.) Start PDFCreator and choose options/directories and and type %PTPath%<Redmon_User>. You should now see in the preview: C:\PDFCreator-Temp\'Preview REDMON_USER'\
