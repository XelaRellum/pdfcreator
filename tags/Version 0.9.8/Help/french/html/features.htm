Aide PDFCreator
 
 

Fontionnalités

  • Créer des documents aux formats suivants : Portable Document Format (PDF), Postscript (PS), Encapsulated Postscript (EPS), PNG, JPEG, BMP, PCX, et TIFF.

  • Protéger vos documents PDF par mot de passe et utiliser un cryptage jusqu'à 128 bits.
