Paramètres PDF Settings : Polices

L'onglet Polices règle les paramètres d'intégration et de substitution des polices dans le PDF.

Intégrer toutes les polices : Lorsque l'option est activée, toutes les polices utilisées dans le document source seront intégrées dans le PDF. Cela permet d'ouvrir le PDF en toute circonstance et de le visionner correctement.

Substituer la police : Lorsque des polices sont intégrées, si la totalité de la police n'est pas utilisée , vous pouvez choisir d'en substituer une partie. Si vous choisissez de substituer des polices, seuls les caractères utilisés seront intégrés au PDF. Cela réduit la taille du PDF. Quand elle est sélectionnée, l'option définit le pourcentage de caractères en-dessous duquel la police doit se situer pour qu'elle soit substituée.
