Aide PDFCreator
 
 

Les documents Word avec un format de page personnalisé sont convertis à la taille standard par défaut

Par défaut, PDFCreator connait seulement les formats de papier existant lors de son installation. Si un format personnalisé est défini dans Word, PDFCreator ne le reconnaît pas et utilise alors le format par défaut.

Pour disposer de nouveaux formats de papier, ils doivent être créés depuis les options Imprimante->Réglages pilote. Ici vous pouvez créer, nommer et dimensionner les formats. Ensuite vous pouvez l'utiliser dans Word.
