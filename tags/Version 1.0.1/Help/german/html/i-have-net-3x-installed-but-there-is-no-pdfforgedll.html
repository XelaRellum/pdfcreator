PDFCreator Hilfe
 
 

Ich habe .Net 3.x installiert, aber finde keine pdfforge.dll

.Net 2 oder höher enthalten leider die .Net 1.1 Funktionen nicht und Microsoft liefert es auch nicht mehr mit windows 7 und Vista aus.

Bitte installieren Sie zusätzlich .Net 1.1 und installieren dann PDFCreator neu. Dann finden Sie die pdfforge.dll im Installationsordner.
