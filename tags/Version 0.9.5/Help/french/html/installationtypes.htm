Aide PDFCreator
 
 

Types d'installation

Il y a plusieurs manières d'installer PDFCreator. Voici quelques informations sur ces différents types d'installation.

Installation standard : L'installation standard installe une imprimante virtuelle pour PDFCreator et permet depuis l'ordinateur de créer des documents PDF, des images Bitmap et des fichiers Postscript.

Installation sur serveur : L'installation sur serveur installe une imprimante réseau qui permet aux ordinateurs connectés à votre réseau de créer des documents PDF, des images Bitmap et des fichiers Postscript. Si vous utilisez ce mode d'installation sur un serveur de type terminal, assurez-vous de ne pas installer PDFCreator en mode application.

Avec une installation sur serveur, vous pouvez choisir le mode Enregistrement automatique pour enregistrer automatiquement les documents créés dans un répertoire spécifique, en utilisant des chemins de type UNC et/ou des suffixes (pour plus d'informations sur le mode d'enregistrement automatique, cliquez ici). Lorsque vous utilisez cette fonctionnalité, assurez-vous que l'utilisateur a les droits suffisants pour enregistrer dans le répertoire spécifié.
Exemple: \\file1\users\<REDMON_USER>\PDFs

Note : Lorsque vous installez PDFCreator sur serveur, assurez-vous de créer un partage de l'imprimante avec les droits d'accès pour vous. Egalement, assurez-vous qu'au-moins une instance de PDFCreator est ouverte. Vous pouvez installer PDFCreator en tant que service en utilisant les outils "instsrv.exe" et "srvany.exe" depuis le kit ressources de votre installation Windows. Si vous installez PDFCreator en tant que service, utilisez un compte utilisateur avec les droits d'accès en écriture dans un répertoire prévu pour l'enregistrement automatique. Les réglages peuvent seulement être changés depuis le serveur. Les utilisateurs ne peuvent modifier aucune option de PDFCreator.
