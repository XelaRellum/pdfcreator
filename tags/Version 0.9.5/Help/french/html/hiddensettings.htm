Aide PDFCreator
 
 

Options cachées

PDFCreator comprend plusieurs paramètres cachés utilisés pour des tests. Ces réglages sont stockés par défaut dans le registre. Il est possible de stocker ces réglages dans un fichier .ini : lancez une nouvelle installation et utilisez l'option /useINI. Si vous avez besoin d'utiliser d'autres paramètres, utilisez l'option /INIFile= parameters avec le programme d'installation. Pour plus d'informations sur les paramètres qui peuvent être utilisés lors de l'installation, utilisez l'option /?.

ClientComputerResolveIPAddress : Dans certains cas spéciaux, il est possible que la variable <ClientComputer> contienne une adresse IP au lieu d'un nom d'hôte. Si vous utilisez cette fonction, PDFCreator essaiera de résoudre l'adresse IP pour trouver un nom dhôte.
Valeurs :
0 - Off (Défaut)
1 - On

DisableEmail : Cette option désactive le bouton d'envoi des e-mails dans PDFCreator.
Valeurs :
0 - Bouton e-mail activé (Défaut)
1 - Bouton e-mail désactivé

PDFCompressionColorCompression*, PDFCompressionGreyCompression* : A l'aide de ces paramètres, il est possible d'affecter individuellement des facteurs de compression, afin de mieux contrôler la compression des images en couleur et en niveaux de gris. Toutes les valeurs doivent être supérieures à zéro.
Valeurs par défaut :
PDFCompressionColorCompressionJPEGMaximumFactor=2
PDFCompressionColorCompressionJPEGHighFactor=0.9
PDFCompressionColorCompressionJPEGMediumFactor=0.5
PDFCompressionColorCompressionJPEGLowFactor=0.25
PDFCompressionColorCompressionJPEGMinimumFactor=0.1
PDFCompressionGreyCompressionJPEGMaximumFactor=2
PDFCompressionGreyCompressionJPEGHighFactor=0.9
PDFCompressionGreyCompressionJPEGMediumFactor=0.5
PDFCompressionGreyCompressionJPEGLowFactor=0.25
PDFCompressionGreyCompressionJPEGMinimumFactor=0.1

PDFOwnerPasswordString : Cette option est utilisée lorsque les options de sécurité PDF sont activées dans PDFCreator. PDFCreator inclue alors automatiquement le mot de passe propriétaire dans le fichier .ini. ATTENTION : Le texte contenu dans le fichier .ini n'est pas crypté. Utilisez cette option à vos risques et périls.

PDFUserPasswordString : Cette option est utilisée lorsque les options de sécurité PDF sont activées dans PDFCreator. PDFCreator inclue alors automatiquement le mot de passe utilisateur dans le fichier .ini. ATTENTION : Le texte contenu dans le fichier .ini n'est pas crypté. Utilisez cette option à vos risques et périls.

RemoveAllKnownFileExtensions : Cette option permet à PDFCreator de supprimer toutes les extensions de fichiers connues en partant de la droite, jusqu'à ce qu'il rencontre une extension inconnue. Par exemple, le nom interne du fichier postscript est 'Pic.jpg.-12.31.2005.pcx'. Le titre devient alors 'Pic.jpg.-12.31.2005'.
Valeurs :
0 - off
1 - on (Défaut)

StandardCreationdate : Cette option permet à l'utilisateur d'inclure une date de création standard dans tous les documents créés par PDFCreator.

StandardDateformat : Cette options permet à l'utilisateur de changer le format de date qu'utilise PDFCreator. Tous les formats de date ne sont pas autorisés. Vérifiez les réglages de votres sytème pour connaître les formats corrects.

StandardKeywords : Cette option permet à l'utilisateur de placer un ensemble de mots clés standard dans tous les documents créés par PDCreator.

StandardModifydate : Cette option permet à l'utilisateur d'inclure une date de modification standard dans tous les documents créés par PDFCreator.

StandardSubject: Cette option permet à l'utilisateur d'inclure un sujet standard dans tous les documents créés par PDFCreator.

StandardTitle : Cette option permet à l'utilisateur d'inclure un titre standard dans tous les documents créés par PDFCreator.
