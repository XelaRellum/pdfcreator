PDFCreator Hilfe
 
 

Häufig gestellte Fragen

Q: Was ist der Unterschied zwischen der AFPL und GNU Version von PDFCreator?

A: Der Hauptunterschied liegt in der benutzten Ghostscript Version. Die AFPL Version ist neuer als die GPL Version, aber die Lizenz schränkt die Benutzung stärker ein. Wenn sie PDFCreator normal benutzen möchten, dann sind beide Versionen in Ordnung, aber die AFPL Version ist vorzuziehen, weil sie neuer und weiter entwickelt ist. Wenn sie PDFCreator auf eine Weise benutzen möchten, die von der AFPL ausgeschlossen wird, dann könnte die GPL Version die gewünschten Rechte mitbringen.

Q: Nach der Installation von PDFCreator bekomme ich den Fehler"Exception EInOutError in module is-XXXXX.exe at 0000631C. File not found". Was kann ich machen?

A: Das ist ein Problem mit alten InstallShield-Installationen, die seit längerem erledigt sein sollten. In den meisten Fällen kann die angegebene Datei einfach gelöscht werden. Wenn sie sich nicht sicher sind, dann können Sie die Datei in einen anderen Ordner verschieben.

Q: Ich dann kann die PDF Datei nicht durchsuchen oder Text daraus kopieren. Was kann ich machen?

A: Das ist ein Problem mit dem Druckertreiber von PDFCreator 0.7.1. Deinstallieren Sie PDFCreator, starten neu und installieren die aktuellste Version.

Q: Ich habe PDFCreator installiert, aber es funktioniert nicht, was kann ich machen?

A: Überprüfen Sie, ob der PDFCreator Drucker installiert ist. Falls nicht, deinstallieren Sie PDFCreator, starten den PC neu und installieren es erneut. Starten Sie den PC nochmal neu und schauen, ob der Drucker installiert ist. Falls nicht, dann schauen Sie in die Setup.log. Falls ein Fehler aufgetreten ist, dann lassen sie es uns im Forum unter SourceForge.net wissen.

Q: Der Druckertreiber ist nicht kompatibel zu einer Richtline auf Ihrem Computer, die NT 4.0 Treiber blockiert.

A: Dieser Fehler tritt auf, wenn eine Gruppenrichtlinie die Verwendung von Kernelmodus Treibern untersagt. Kernelmodus Treiber haben Zugriff auf den Systemspeicher und schlechte Treiber können gravierende Probleme hervorrufen. So können die Treiber erlaubt werden:

  1. Drücken Sie auf Start und dann Ausführen
  2. Geben Sie gpedit.msc ein und drücken ENTER
  3. Navigate to Computerkonfiguration/Administrative Vorlagen/Drucker und ändern Sie "Installation von Druckern, die Kernelmodustreiber verwenden, nicht zulassen" auf Deaktiviert.

Q: Warum erscheint schwarzer Text leicht gräulich?

A: Wenn die Farben als CMYK gespeichert werden, wird der Text leicht gräulich, obwohl der Text 100% schwarz ist. Abhilfe schafft es, das Farbmodell in den PDF Einstellungen auf RGB zu setzen..

Q: Unterstützt PDFCreator mein Programm XYZ?

A: PDFCreator unterstützt alle Programme, die über die Windows Drucker drucken können.
