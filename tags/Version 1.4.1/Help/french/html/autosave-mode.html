Aide PDFCreator
 
 

Enregistrement automatique

L'option Enregistrement automatique permet à l'utilisateur de créer des fichiers PDF automatiquement avec des paramètres prédéfinis, et de l'enregistrer à un emplacement prédéterminé. Quand l'option Enregistrement automatique est activée, PDFCreator ne demande aucun nom de fichier ni de chemin d'enregistrement. Voici la liste des paramètres qui doivent être définis pour utiliser l'enregistrement automatique :

  • Format d'enregistrement automatique : C'est le format dans lequel vous souhaitez que PDFCreator enregistre les documents convertis.
  • Filename : Définit le nom du fichier pour tous les documents créés. Les variables <DateTime> incluent la date et l'heure de création dans le nom du fichier.
  • Ajouter un suffixe au nom de fichier : Le suffixe est une autre variable ajoutée aux nom des documents. Voici la liste des suffixes utilisables :
    • <Author> - Nom de l'auteur du document
    • <ClientComputer> - Nom de l'ordinateur ou adresse IP du client qui veut créer le PDF (à utiliser en cas d'installation sur serveur). ll peut aussi être utilisé dans le chemin d'enregistrement automatique pour sauver directement sur le poste client (i.e. "\\<ClientComputer>\c$\files").
    • <Computername> - Nom de l'ordinateur.
    • <Counter> - Un compteur allant de 1 à 922.337.203.685.477.
    • <DateTime> - Date et heure auxquelles le PDF est créé.
    • DocumentFilename - Si <Title> correspond à un nom de fichier, cette variable contiendra un nom de fichier.
    • DocumentPath - Si <Title> correspond à nom de fichier complet, cette variable contiendra le chemin. Si cette variable est utilisée pour le répertoire, tous les fichiers seront enregistrés dans le même répertoire que le fichier imprimé.
      Note : Seules quelques applications (telles que Autocad ou Irfanview) utilisent les noms de fichiers complets. C'est pourquoi ce suffixe ne doit être utilisé sans examiner l'application.
    • <JobID> - Le numéro de tâche, représenté par un entier.
    • <PrinterName> - Le nom de l'imprimante, généralement "PDFCreator".
    • <SessionID> - L'identifiant de la session client de Windows Terminal Server. Cette valeur sera égale à zéro si Windows Terminal Server n'est pas utilisé.
    • <Title> - Le nom de la tâche d'impression (ex : "Page de test").
    • <Username> - Nom d'utilisateur de la session Windows en cours.
  • Utiliser ce répertoire pour l'enregistrement automatique : Ce paramètre optionnel permet de définir un répertoire pour l'enregistrement automatique. Les sufixes suivants peuvent être utilisés dans la définition du répertoire :
    • <MyFiles> - Chemin du dossier Mes Documents de l'utilisateur en cours.
    • <MyDesktop> - Chemin vers le dossier Bureau de l'utilisateur en cours.
    • <Temp> - Chemin vers le dossier temporaire de l'utilisateur en cours.
  • Ouvrir le document avec l'application par défaut, après l'enregistrement automatique : Après l'enregistrement automatique, le document converti est ouvert par l'application par défaut associée à son format.
  • Envoyer un e-mail après l'enregistrement automatique. Une fois le document converti et enregistré, PDFCreator envoie le fichier converti par e-mail.

Suffixes RedMon
Les versions de PDFCreator antérieures à la version 1.3.0 utilisaient RedMon comme moniteur d'impression. A partir de la version 1.3.0, cela a été modifié et les suffixes basés sur l'environnement RedMon ne sont plus valides. Ils ont été remplacés par les suffixes correspondants comme <JobID> ou <ClientComputer>.

Note: Si vous souhaitez utiliser les paramètres de sécurité PDF dans le mode d'enregistrement automatique, merci de consulter les paramètres cachés pour savoir comment attribuer un mot de passe.
