SVG

Dieser Menüpunkt erlaubt Einstellungen für das SVG-Format vorzunehmen.

Dieses Format befindet sich noch im Experimentierstadium und liefert unter Umständen nicht die erwünschten Ergebnisse.

Auflösung: Einstellen der DPI der Ausgabedatei. Je höher die DPI Zahl ist, um so detailierter, aber auch um so größer wird die Datei. Der Standardwert ist 72.
