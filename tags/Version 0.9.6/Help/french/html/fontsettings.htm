Aide PDFCreator
 
 

Polices

Cette page vous permet de choisir les polices utilisées dans les fenêtres et légendes de PDFCreator. Par défaut, PDFCreator utlise les polices Windows de votre ordinateur, mais vous pouvez les changer pour n'importe quelle police de votre ordinateur. Un bouton de test permet de tester la police sans la sélectionner réellement.
