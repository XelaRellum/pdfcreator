Aide PDFCreator
 
 

Impression

Si vous activez cette option, PDFCreator imprime le document après l'avoir converti.

Imprimer après l'enregistrement : Lance l'impression après avoir enregistré le document.

Imprimante : Sélectionnez l'imprimante à utiliser.
Exemples :
"HP4300"
"\\printserver\HP4300"

Demander à l'utilisateur : Ces options permettent à l'utilisateur d'afficher ou non la fenêtre d'impression.
Les valeurs possibles sont :

  • Non (Default)
  • Afficher le fenêtre d'impression standard
  • Afficher la fenêtre de propriétés de l'imprimante
  • Sélectionner l'imprimante par défaut sans intervention de l'utilisateur

Cacher la barre de progression pendant l'impression : Cette option montre ou cache la barre de progression pendant l'impression.

Imprimante duplex : Active l'impression recto-verso. Vous pouvez aussi activer le pivotement d'une page sur deux pour des problèmes de reliure.
