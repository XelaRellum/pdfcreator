Paramètres PDF : Signature

Signer les fichiers PDF permet au destinataire de vérifier que le document a bien été rédigé par l'auteur indiqué et qu'il n'a pas été modifié après la signature. Ce système est basé sur un mécanisme de clé publique/clé privée. La clé privée est utilisée pour chiffrer une empreinte (hash) du document. Cette empteinte cryptée peut être décryptée à l'aide de la clé publique, pour ensuite être comparée avec une empreinte du document à vérifier. Les deux empreintes correspondent uniquement lorsque les deux clés publique et privée vont de paire, et lorsque le document n'a pas été modifié.

Pour signer numériquement un fochier PDF, plusieurs éléments sont nécessaires :

  1. Un certificat numérique au format PFX\P12 permettant de signer des fichiers.
  2. .Net 1.1 doit être installé. Ceci entraîne que la signature de documents PDF n'est pas possible sous Windows 95 et Windows NT 4 Workstation, puisque .Net 1.1 n'est pas supporté pour ces systèmes.
  3. Le fichier pdfforge.dll doit être présent dans le répertoire d'installation de PDFCreator dans PlugIns\pdfforge. Si .Net 1.1 est installé, cette DLL sera automatiquement installée en même temps que PDFCreator.

Fichier Pfx/P12 : Le fichier contenant la signature. Il contient la clé privée utilisée pour crypter l'empreinte du PDF à signer.

Raison signature : La raison de la signature. C'est un texte court qui indique pourqui le fichier est signé numériquement.

Contact signature : Le nom de la personne qui a signé le fichier.

Localisation signature : Endroit (Pays, ville, etc.), où le fichier a été signé.

Position signature : Permet à Acrobat Reader d'afficher un icône qui indique la validité de la signature. Les coordonnées de cet icône peuvent être réglées grâce aux valeurs X/Y gauche et X/Y droite.

Signature multiple autorisée : Permet la signature ultérieure du fichier par d'autres personnes.

Vous pouvez obtenir des certificats gratuits à l'adresse http://www.cacert.org. Notez que ce n'est pas un service où un simple enregistremetn suffit. CACert est basé sur un réseau de confiance, au sein duquel vous êtes évalué par d'autres personnes, et dans lequel vous pouvez évaluer d'autres personnes. Cela nécessite un peu de travail pour récolter suffisamment de points afin accéder à un statut "de confiance", mais c'est réellement gratuit et nous vous encourageons à essayer.
