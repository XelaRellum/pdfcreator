EPS Einstellungen

Hier werden die Einstellungen für das EPS (Encapsulated PostScript) Format vorgenommen. EPS ist ein Standard für Druck und Grafik.

Sprachversion: Legt die Sprachversion des Formates fest. Für den Druck wird manchmal eine spezielle Version vorausgesetzt. Der Standardwert ist 2.
