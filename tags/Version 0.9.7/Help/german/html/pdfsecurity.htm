PDF-Einstellungen: Sicherheit

Das Sicherheits Menü setzt PDF-Sicherheitseinstellungen für die Verschlüsselung von PDF-Dokumenten und ermöglicht die Deaktivierung von PDF-Merkmalen für mehr Sicherheit in PDF-Dokumenten.

Sicherheit benutzen: Aktiviert und deaktiviert die Sicheheitseinstellungen in PDFCreator.

Verschlüsselungsprogramm: PDFCreator unterstützt 2 verschiedene Verschlüsselungsmöglichkeiten. Es ist einmal direkt mit Ghostscript ab der Version 8.14 und darüber möglich. Oder mit dem enthalten Programm pdfenc. Beide unterstützen die erweiterten Sicherheitsmerkmale mit Passwortschutz und 128 Bit Verschlüsselung.

Verschlüsselungsgrad: Eine Verschlüsselung verhindert unberechtigtes Kopieren von in PDF-Dokumenten enthaltenen Informationen. PDFCreator bietet 2 Verschlüsselungstufen: Gering (40 Bit) und hoch (128 Bit). Eine hohe Verschlüsselungsstufe ist nur mit einer PDF-Version 1.4 möglich.

Passwörter: Sie können einmal ein Besitzerpasswort und/oder ein Benutzerpasswort einstellen. Mit Hilfe des Besitzerpasswortes verhindern Sie die Bearbeitung von PDF-Dokumenten. Ein Benutzerpasswort verhindert das nicht erlaubte Öffnen von PDF-Dokumenten.

Verweigere Benutzern: Mit diese Einstellungen verbieten Sie Drucken, kopieren von Text und Bildern, das Verändern von Dokumenten und das Hinzufügen und Verändern von Kommentaren.

Erweiterte Berechtigungen: Diese Einstellungen können nur bei 128 Bit Verschlüsselungen gesetzt werden. Es können die Berechtigungen für das Drucken in niedriger Auflösung, das Ausfüllen von Formularen, das Erlauben von Bildschirmlesern und die Änderung der Anordnung gesetzt werden.
