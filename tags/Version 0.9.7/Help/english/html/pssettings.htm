PS Settings

This menu changes the settings when converting to Postscript (PS) format. The Postscript format is a standard for printing and imaging.

Language Level: Sets the language level or version to use when converting a document. If you are making a Postscript file to be used for printing the printer may specify which level of Postscript it can handle. The default Language Level is 2.
