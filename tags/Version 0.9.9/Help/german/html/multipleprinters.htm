PDFCreator Hilfe
 
 

Mehrere Drucker benutzen

PDFCreator kann mit mehreren Druckern verwendet werden. Die Einstellungen dazu erreichen Sie im Hauptfenster unter Drucker -> Drucker.

Drucker Erstellen, Löschen

Sie können PDFCreator dazu benutzen, neue Drucker anzulegen. Diese Drucker sind spezielle Drucker die für die Zusammenarbeit mit dem Programm PDFCreator eingerichtet wurden und nicht für einen Ausdruck auf Papier gedacht sind. Geben Sie dazu einen Namen für den Drucker ein und drücken auf "Drucker hinzufügen". Wählen Sie einen Drucker aus und drücken auf "Drucker löschen", um ihn zu entfernen.

Anmerkung: Sie müssen Administratorrechte haben, um Drucker anlegen zu können. Ab Windows Vista muss PDFCreator außerdem als Administrator ausgeführt werden.

Einen Drucker mit einem Profil verbinden

Sie können jeden Drucker mit einem Einstellungsprofil verknüpfen. Sie können beispielsweise einen Rechnungsdrucker haben, der die Rechnungen automatisch als PDF in einem bestimmten Verzeichnis ablegt, und einen zweiten für allgemeine Aufgaben.

Weiter Möglichkeiten sind:

  • JPEG-Drucker um Bilddateien zu erhalten
  • hochauflösende PDFs für den Ausdruck
  • PDFs in niedriger Auflösung für eMails
  • ein PDF-Drucker bei dem ein Standard-Autor, Standard-Passwort usw. festgelegt wurde

Um zu erfahren, wie Sie Profile einrichten, schauen Sie bitte ins Kapitel "Profile benutzen".
